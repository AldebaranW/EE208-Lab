中超争冠悬念"人为"倾斜?马尔康停赛1场 泰山不战3-0用微信扫码二维码分享至好友和朋友圈网易体育12月26日报道：今天，官方发布公告，宣布原定于12月27日进行的三场中超第33轮比赛因为有球队弃权，直接判定对手3-0获胜，其中一场是山东VS梅州客家，梅州申请弃权，泰山3-0获胜。这样，在中超争冠形势极度胶着的情况下，山东兵不血刃的拿下3分，还斩获3个净胜球，多赛1场的情况下在积分榜上3分领跑，与三镇的净胜球差距也只有3个了。上轮中超，争冠的山东泰山和武汉三镇都输了球，山东泰山0-2完败上海海港，武汉三镇则是在客场0-1被蓉城绝杀，两队在比赛过程中都遭遇了非常有争议的判罚。山东与海港的比赛，下半场陈蒲一度为山东扳平比分，但却被吹罚越位在先无效，但VAR并没有给出清晰的越位镜头，赛后山东队的球队官员围攻主裁判张雷。开始结束用这种方式输掉比赛后，三镇的当家射手马尔康在返回更衣室的途中，用脚踢坏了赛事入场拱门，中国足协官方今天公布了对马尔康的处罚，停赛1场，罚款1万。据了解，三镇在与成都比赛后就和中国足协相关方面做了沟通，并承认错误，态度非常诚恳，但这并没有让马尔康免于被罚。目前，马尔康已经打进27球，在射手榜上排名第1，是三镇锋线的头号杀器，他的缺席对三镇来说是非常大的损失。第33轮，三镇将在主场迎战浙江队，浙江队目前排名第3，非常有希望冲击一个亚冠名额，而且在两队首回合交手中，浙江队就取得了胜利，所以对三镇来说，这是一场硬仗，如果赢不下来，冠军很可能就没了，这对本赛季一直领跑的三镇来说将会是非常大的打击。作为争冠的直接对手，上轮同时输球，在还有两轮的情况下，一支球队对手弃权，一支却要面临一场硬仗，境遇天差地别。12月4日，中超第29轮，山东与深圳的比赛被迫延期，补赛在12月19日进行，但那时内已经无法凑齐参赛的球员。据了解，当时深圳队想放弃这场比赛，但却未获得官方同意，最终年轻球员出战的深圳队0-8惨败，那一场让山东追回了8个净胜球，与三镇的净胜球差距缩小到了5个。要知道，如果深圳队被允许弃赛，那么也就是0-3负，三镇在净胜球上依旧会有非常大的领先，而在三镇和泰山同分的情况下，两队本赛季主客场交手又都是1-1，净胜球很可能会决定最后冠军的归属，所以，弃赛不弃赛，其实区别是非常大的。而对中国足协和中超官方来说，一会允许球队弃赛，一会又不允许弃赛，毫无标准可言，在中超收官这么关键的时刻，无疑起到了非常糟糕的作用。在中超进行到倒数第2轮的情况下，在调整了补报球员规则的情况下，依然还有多支球队弃赛，中超联赛的组织和管理备受外界质疑，有网友就表示：“联赛就像闹剧，笑话一样。”。当下的中国足球，欠薪遍地，俱乐部朝不保夕，一地鸡毛，唯一还有点噱头的冠军之争，在政策、赛程调整之下，被搞的稀碎，毫无公平公正可言，这对整个中国足球环境的打击是致命的，希望中国足球的管理者们，好好反思反思吧！爱讲故事的
