国际露营大会走进千岛湖 从"心"发现乐享情境_网易体育
2015-09-11
国际露营大会走进千岛湖 从"心"发现乐享情境用微信扫码二维码分享至好友和朋友圈2015年9月19日至20日，退去盛夏的炽热，奥迪2015中国国际大会带着“心”的发现再次走进，数万名户外露营者将在此尽享户外露营运动带来的全新乐境。神秘明星、彩色跑、湖畔露营、房车体验、家庭户外体验营、山地半程马拉松等。据介绍，中国国际露营大会由国家体育总局批准，国家体育总局登山运动管理中心、浙江省体育主办，一汽-大众奥迪冠名赞助，北京露营之家体育文化传播有限公司、淳安县人民政府政府承办，是目前最专业、最权威的休闲露营全民健身赛事活动。千岛湖，以千岛、秀水、金腰带（岛屿与湖水相接处环绕着有一层金黄色的土带，称之名“金腰带”）为主要特色景观，景区总面积982平方公里，是中国面积最大的森林公园，尽享天奢自然，贵胄之气。为了让参会选手更好地“阅读”千岛湖，赛会活动将在两天时间里通过“明星领跑彩色马拉松十分赛”、“山地半程马拉松”、“家庭户外体验营”、“全新A6 allroad quattro湖畔嘉年华”等活动，与 “秀水广场”、“芳菲隐珠阳光体育公园”、“天屿山”、“牧心谷”等千岛湖独具代表特色的景致相融入。通过这样精彩纷呈的活动，参会选手不仅可以全方位领略千岛湖的致美，更可以多维度地感悟户外休闲露营运动所带来的激情与活力。中国国际露营大会作为国内举办规模最大的露营大会，将休闲露营运动与泛户外旅游深度融合，其倡导的健康乐活方式与奥迪“进取 尊贵 动感”的品牌内涵高度契合。一汽-大众奥迪与本次露营大会的成功携手，即是以进取精神开启国内“汽车露营”新风尚，大力倡导“自然、和谐、健康”绿色生活方式，与参会选手们一同分享时尚健康新生活的体现。为了让奥迪用户及广大户外运动爱好者享受更为丰富的户外运动体验，近距离感受奥迪“进取 尊贵 动感”的品牌价值与卓越的科技感，一汽-大众奥迪特别安排了全新奥迪A6 allroad quattro在此次赛会的亮相，为旅行和旅行车的爱好者们送上意外的惊喜。赛会期间，全新A6 allroad quattro湖畔嘉年华、展示和试乘试驾等精彩活动将纷纷举行，为户外运动爱好者带来更多的动感体验。据悉，开幕式现场，众多活动跃在体育界、主持界以及文艺界的大腕将加入到此次露营狂欢的阵容之中。他们将与户外露营爱好者共同分享交流户外露营心得并一同参与到情侣跑、亲子跑、越动跑的行列之中。为了减少紧张刺激的户外运动所带来身体疲惫，奥迪2015中国国际露营大会千岛湖站特别为露营爱好者们安排了“全新A6 allroad quattro湖畔嘉年华”。在结束紧张的身体运动后，参会者们将跟随着音乐的节奏，将所有的压力和烦恼驱散。任何一项运动的乐趣绝不仅是旁观者的“观看”之悦，亲身体验参与其中才是悦境的真谛。奥迪2015中国国际露营大会千岛站各项赛会活动均设置了全民皆可参与的“悦享赛”，充分将参与之悦做足、做透。其中，以亲情为主题的家庭户外体验营将成为户外爱好者享受阳光般亲情的不二之选。来自全国的20个团队将参与其中，共同打造一期属于自己家庭的《爸爸去哪儿》。家庭户外体验营将通过亲子足球、亲子篮球、亲子皮划艇、八大行星探秘等一系列重在“体验之悦”的露营互动环节，让孩子们在虚拟的太空农场与父母共同中探索生命的奥秘。搜狐汽车
