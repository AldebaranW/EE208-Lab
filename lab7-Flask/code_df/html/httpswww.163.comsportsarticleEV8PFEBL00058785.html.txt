乐跑5公里领略千年城变  全程零废弃“一杯到底”|永庆|马拉松|跑者|海心沙_网易体育
2019-11-30
乐跑5公里领略千年城变  全程零废弃“一杯到底”用微信扫码二维码分享至好友和朋友圈网易体育11月30日报道：“穿越”坊，畅览，领略广州旧城与新貌，只要跑个5公里？没错！11月30日下午，“2019广州城市赛” 在广州海心沙鸣笛开跑，它是深受广州市民热爱的健身运动嘉年华，已经连续举办7年，今年活动亮点纷呈：献礼新中国成立70周年快闪唱红歌；永庆坊遇见广州塔展露千年古城蝶变魅力；突出科技引领，倡导绿色低碳，参赛选手全程佩戴专业计时芯片，现场设置了分类垃圾桶实现全程零废弃。据了解，该次乐跑赛由国家体育总局群众体育司、广东省生态环境厅指导，万科集团主办，广东广播电视台协办，共青团广东省委员会和广东省体育基金会是活动支持单位。据统计，7年来，累计707个单位，34491人参与广州城市乐跑赛，跑步里程长达172455公里，而今年乐跑赛吸引了104家参赛单位，4000多名参与。“好有趣的赛场！站在永庆坊，望见小蛮腰。”跑者Jolin刚一进场就雀跃不已，她自小在西关长大，永庆坊牌坊、李小龙故居、八和会馆、金声电影院都是儿时记忆的重要部分，热心的她，一边寻找最美角度留影，一边给其他跑友当导游。令跑友们感到惊喜的，不仅仅有永庆坊与广州塔的相遇，还有城市时光展，这场展览浓缩广州城市发展历程，重点展示新中国成立以来广州城市面貌的变化。“我们走着，跑着，一转身，一抬头，都能感受到历史的沉淀和文化的生机，这个赛场的布置真走心！”跑友林先生说着竖起大拇指，他自称新广州人，大学毕业后留在广州工作，结婚生子，他说，“跑步，不纯粹是运动，边跑，边看，边感受，是一个融入的过程，我就是跑着步，感受着这座城市的变化，爱上她，而且越来越爱。”乐跑赛是一场全民健身嘉年华。开跑之前，从永庆坊牌坊传来一把清脆童声：“我和我的祖国，一刻也不能分割。”伴着熟悉的旋律，人们纷纷聚拢应和，高声唱起《我和我的祖国》，一时间，开跑现场变成一场4000多人一起参与的快闪合唱，大家情绪昂扬地唱着，有人手牵手做起人浪，有人跟随节奏挥舞毛巾、帽子，有人高高举起标语：“我爱广州”。记者从乐跑赛现场了解到，今年活动两大亮点分别是：科技助力和绿色环保。据介绍，今年参赛选手全程佩戴马拉松专业计时芯片，完赛成绩不受参赛项目起跑先后影响。而在绿色环保方面，主办方倡导“一杯到底”全程“零废弃”，鼓励跑者全程只使用一个纸杯，活动现场还设置零废弃展台和分类垃圾箱，引导跑者分类投放垃圾，实现乐跑全程“零废弃”。此外，本次乐跑赛认购了1000千瓦时绿色风电，将达成100%绿色电力消费效果，相当于减排二氧化碳900.70千克，二氧化硫0.47千克，氮氧化物0.43千克。据介绍，2019广州城市乐跑赛设立三项团体大奖，包括：团队成绩冠军、亚军、季军，以及最佳风尚奖、最资深团队奖。最终，广州白云国际机场股份有限公司团队以21分13秒的优异成绩夺得团体第一名，中国南方航空股份有限公司与合富辉煌（中国）房地产顾问有限公司分别夺得团体赛第二、三名。2019年广州城市乐跑赛是2019年城市乐跑赛的最后一站。城市乐跑赛是在国家体育总局指导下，由万科集团发起并主办的城市跑步活动，是国家践行全国健身战略的重点扶持项目之一。该赛事以“跑出快乐、跑出健康、跑出友谊”为宗旨，属于非商业性、非竞技性群众体育活动，自2013年创办至今，城市乐跑赛足迹遍布全国60个城市，举办了超过380场赛事，参与总人数80万人次，共有2万多家企事业单位、4万多个家庭及51所高校参与，目前已经发展成为包括全程马拉松、社区乐跑赛、大学乐跑赛在内的乐跑全系列活动，还获得过“中国最佳健康创新实践奖”等多个奖项。谈资成都
