范子铭与首钢合练：一个半月来第一次摸球 真没手感用微信扫码二维码分享至好友和朋友圈新京报讯9月1日晚，随中国男篮结束亚洲杯任务的几经辗转回到。今日上午，他就归队加入男篮的合练。接受新京报记者独家采访时，小范直言重新回到首钢大家庭，时隔一个半月再摸到篮球非常兴奋，“我目前最强烈的想法就是别给球队拖后腿。”范子铭7月31日抵达中国香港，由于防控和机票等原因，直到8月24日才从四川入境，经过7+3隔离后，昨晚终于返京。刚到首钢体育大厦，他就向主教练解立彬报到并进行长谈。“能从聊天中听出解指导对球队包括我有很多新的想法。”小范告诉新京报记者，因为当下的身体状况还达不到为球队做点什么，所以交流之后最大的想法就是“给自己上上强度，争取把身体恢复过来”。9月2日上午，范子铭重回训练场，跟队练了投篮和力量。这是他一个半月以来第一次摸球。“很长时间只能在房间做徒手练习，都快不知道篮球是圆的还是方的了。”这期间，小范在无数个晚上想过归队合练的场景。当这一天终于到来，他直言兴奋，也感受到阔别近两个月，全队上下有了更多年轻人的朝气。许久没有系统训练，体能肯定会有欠缺，范子铭也着急。距新赛季开赛仅剩一个月，想要追上全队的训练进度时间紧、任务重，但他还是很有信心。队长归队时曾表示，经过与教练组沟通，需要5天到一周先恢复身体。小范透露，川队实际只用了3天左右就跟队正常训练了，“我当然希望越快越好，但还是要循序渐进。”首钢队下午的对抗，范子铭还无法加入，就约了助理教练李昂，提前一小时到球馆先进行有球训练。有的练习项目，小范边做边感叹“真是没手感”；从喘气的程度也能看出，他的体能确实还需要加强。首钢目前已经恢复了一天两练，小范没赶上跟大部队一天三练，于是便主动加练。“也是心里着急嘛，能感受到累总比空洞地等待，不知道什么时候能系统训练要好，所以现在加班，一点不算什么。”他说。历史名匠锁 2023-11-01 09:47:31点点娱乐说说 2023-11-01 09:43:40历史名匠锁 2023-11-01 10:37:57篮球侍郎 2023-11-02 00:05:27乐悦侃娱乐 2023-11-01 00:06:26弄月公子 2023-10-28 12:07:55皮多爱体育 2023-10-31 23:06:24聊史 2023-11-01 10:34:02德融视讯 2023-11-01 13:52:10东球弟 2023-11-01 21:43:09篮球侍郎 2023-11-01 01:05:03狼叔评论 2023-11-01 01:33:19青岛文艺 2023-11-01 14:55:51中国新闻周刊 2023-11-01 13:38:46东球弟 2023-11-01 08:52:34理工男评篮球 2023-11-01 22:01:44最爱生活汇 2023-11-01 12:46:34财联社 2023-11-01 09:55:27侃球熊弟 2023-11-01 09:37:23新娱乐圈看点 2023-11-01 10:17:54网易体育 2023-11-01 21:37:10后厂村体工队 2023-11-01 09:35:41东方豪侠 2023-11-01 08:11:10东球弟 2023-10-30 12:50:05网易体育 2023-11-01 12:36:21不掉线电波 2023-11-01 14:29:21牛锅小钒 2023-10-31 09:43:26财联社 2023-11-01 15:20:38澎湃新闻 2023-11-01 20:27:13网易体育 2023-11-01 21:48:58央视新闻客户端 2023-11-01 15:59:20静海融媒 2023-11-01 14:57:48星晨视频 2023-11-01 15:02:49澎湃新闻 2023-11-01 16:06:54滑县市场监管 2023-11-01 20:50:41中国海警 2023-11-01 13:17:44财联社 2023-10-31 22:10:49财联社 2023-11-01 15:43:45央视新闻客户端 2023-11-01 13:27:42懂球帝 2023-11-01 05:37:33
