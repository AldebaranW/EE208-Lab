京媒：许梦君侯逸凡互换 俞长栋李根将加盟北控用微信扫码二维码分享至好友和朋友圈本报讯（记者 刘艾林）在结束了2019-20赛季的征程之后，休赛期短暂的北控男篮并没有闲着，而是加紧在位于亦庄弘赫国际体育中心的训练馆内进行新赛季开赛前的集训。这几天球队也是多了几张“新面孔”，近一段时间内与球队传出转会绯闻的几名联盟中的老将、都已到队，还有最新消息涉及到的锋线球员等，也都加入到北控队的训练中。StartFragment人都聚齐了，而北控俱乐部就是迟迟不官宣，这也是俱乐部一贯谨慎态度的体现。8月27日，苏州肯帝亚队男篮总经理史琳杰在社交媒体上写道：“君侯相易，有梦不凡！欢迎小许，祝福猴子，希望在新的环境都能重新证明自己！”这算是证明了该队与北控男篮所做的一次人员交换属实。这里的“君侯”分别指的是北控男篮的小前锋、投手，以及肯帝亚队队员侯逸凡。这位与中国的女子国际象棋特级大师同名的侯逸凡，是一名在江苏队打了多个赛季的中生代球员，今年28岁。最近两个赛季，他的表现总体是不错的。上赛季，侯逸凡场均9.6分。而本赛季他场均拿下7.1分。除了有一定的三分球准确性外，他冲击篮筐也很有侵略性。本赛季在对阵队时，侯逸凡曾经砍下29分。考虑到投手许梦君本赛季在球队的位置很不稳定，几乎罕有上场时间，给球队提供的火力有限，所以交换来侯逸凡也可以给北控队提供更多的帮助。除了侯逸凡之外，球迷最为关心的当然是此前曾效力于首钢男篮并获得过总冠军，但本赛季因受伤在队表现不佳的李根，以及队的内线球员俞长栋，这两位老将战斗力仍然不弱。这几日两人一同出现在北控训练场，据北京青年报记者了解，俞长栋与北控的合同将会很快确定，李根的加盟事宜则会稍后进行。再加上与上海队交换来的后卫球员罗旭东，北控男篮等于在内线、锋线、后卫线这三线上都有补强。今年的特殊情况造成了新旧两个赛季之间的间隔极短，只有两个月左右的阵容调整和转会运作时间，北控俱乐部在如此短暂的时间内做了大量的工作，进行了最大限度的阵容补强，已经殊为不易了。EndFragment篮球资讯达人 2023-11-01 22:09:41德融视讯 2023-11-01 13:52:10看度新闻 2023-11-01 15:38:52东球弟 2023-11-01 21:43:09青岛文艺 2023-11-01 14:55:51篮球资讯达人 2023-11-01 23:12:04东球弟 2023-11-01 08:52:34理工男评篮球 2023-11-01 22:01:44侃球熊弟 2023-11-01 09:37:23财联社 2023-11-01 09:55:27新娱乐圈看点 2023-11-01 10:17:54网易体育 2023-11-01 21:37:10后厂村体工队 2023-11-01 09:35:41东方豪侠 2023-11-01 08:11:10东球弟 2023-10-30 12:50:05网易体育 2023-11-01 12:36:21澎湃新闻 2023-11-01 20:27:13财联社 2023-11-01 15:20:38网易体育 2023-11-01 21:48:58央视新闻客户端 2023-11-01 15:59:20静海融媒 2023-11-01 14:57:48澎湃新闻 2023-11-01 16:06:54星晨视频 2023-11-01 15:02:49滑县市场监管 2023-11-01 20:50:41中国海警 2023-11-01 13:17:44财联社 2023-10-31 22:10:49财联社 2023-11-01 15:43:45央视新闻客户端 2023-11-01 13:27:42不掉线电波 2023-11-01 14:29:21新京报 2023-11-01 18:14:13直播吧 2023-11-01 19:20:20不掉线电波 2023-11-01 21:44:48网易体育 2023-11-01 21:31:49懂球帝 2023-11-01 05:37:33
