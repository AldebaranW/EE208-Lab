CBA向广厦开罚单：王博停1场罚5万总经理停1场2万|广厦队_网易体育
2022-10-31
CBA向广厦开罚单：王博停1场罚5万总经理停1场2万用微信扫码二维码分享至好友和朋友圈网易体育10月31日报道：时间10月31日晚上，CBA公司向队的主教练及总经理缪寿守开出罚单，王博被停赛1场罚款5万，缪寿守停赛1场罚款2万。处罚的原因是：昨晚与广州队的常规赛第四节剩余3分46秒时，主教练王博对裁判员判罚不满且情绪激动，在其本人被宣判取消当场比赛资格、离开场地前，有摔砸战术板的不当行为。广厦队总经理缪寿守违规进入比赛场内，干扰赛场秩序。王博和缪寿守分别作为球队教练组及俱乐部联赛赛风赛纪的第一责任人，上述不当行为给联赛造成了负面影响。目前，CBA常规赛进入休赛季，二人的停赛将在第二阶段开始的第一场比赛执行。常规赛第一阶段，广厦队受到很大的伤病困扰，胡金秋、外援卡明斯、赵岩昊都收到伤病困扰，广厦队取得了4胜5负的成绩，排名第15位。中篮联〔2022〕342号各CBA俱乐部：2022年10月30日，2022-2023赛季CBA联赛常规赛第九轮场序90，在广厦雄狮俱乐部东阳光鲜虫草篮球队（简称“广厦队”）与广州龙狮俱乐部龙狮篮球队的比赛进行到第四节剩余3分46秒时，广厦队主教练王博对裁判员判罚不满且情绪激动，在其本人被宣判取消当场比赛资格、离开场地前，有摔砸战术板的不当行为。同时，广厦队总经理缪寿守违规进入比赛场内，干扰赛场秩序。王博和缪寿守分别作为球队教练组及俱乐部联赛赛风赛纪的第一责任人，上述不当行为给联赛造成了负面影响。为严肃联赛纪律，根据《中国男子篮球职业联赛纪律准则》（简称《纪律准则》）第二章第二十三条之规定，对广厦队主教练王博处以罚款人民币5万元、停赛1场的处罚，停赛场次为2022-2023赛季CBA联赛常规赛第十轮广厦队的比赛。根据《纪律准则》第二章第二十八条之规定，对广厦队总经理缪寿守处以罚款人民币2万元、停赛1场的处罚，停赛场次为2022-2023赛季CBA联赛常规赛第十轮广厦队的比赛。以上人员罚款的缴纳按照《纪律准则》第一章第十三条之规定办理。特此函告。中篮联（北京）体育有限公司2022年10月31日环太平洋老正太 2023-11-01 22:10:56小绿说故事 2023-11-01 22:37:40篮色生活 2023-11-01 23:15:15讲坛 2023-11-01 11:18:59小绿说故事 2023-11-01 22:37:58薇说体育 2023-11-01 23:12:06篮球实战宝典 2023-10-30 10:09:27篮球资讯达人 2023-11-01 22:06:26阳光玩笑君 2023-11-01 11:00:29德融视讯 2023-11-01 13:52:10看度新闻 2023-11-01 15:38:52东球弟 2023-11-01 21:43:09环太平洋老正太 2023-11-01 22:02:10青岛文艺 2023-11-01 14:55:51篮球资讯达人 2023-11-01 23:12:04东球弟 2023-11-01 08:52:34理工男评篮球 2023-11-01 22:01:44网易体育 2023-11-01 21:37:10财联社 2023-11-01 09:55:27侃球熊弟 2023-11-01 09:37:23新娱乐圈看点 2023-11-01 10:17:54后厂村体工队 2023-11-01 09:35:41樱桃没有丸子 2023-11-01 15:58:28东方豪侠 2023-11-01 08:11:10东球弟 2023-10-30 12:50:05网易体育 2023-11-01 12:36:21不掉线电波 2023-11-01 14:29:21澎湃新闻 2023-11-01 20:27:13财联社 2023-11-01 15:20:38网易体育 2023-11-01 21:48:58央视新闻客户端 2023-11-01 15:59:20静海融媒 2023-11-01 14:57:48星晨视频 2023-11-01 15:02:49澎湃新闻 2023-11-01 16:06:54滑县市场监管 2023-11-01 20:50:41中国海警 2023-11-01 13:17:44财联社 2023-10-31 22:10:49财联社 2023-11-01 15:43:45央视新闻客户端 2023-11-01 13:27:42懂球帝 2023-11-01 05:37:33
