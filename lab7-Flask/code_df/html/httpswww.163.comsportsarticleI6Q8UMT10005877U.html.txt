穆雷：没有摧毁我的 会让我更强大|安迪·穆雷|科比|约基奇|纳什_网易体育
2023-06-09
穆雷：没有摧毁我的 会让我更强大用微信扫码二维码分享至好友和朋友圈总决赛G3终场前3分钟，贾马尔-穆雷在左侧三分线外运球，上前掩护，这是他们早在9年前就开始实践的战术，已经形成了肌肉记忆。穆雷利用掩护加速推进，阿德巴约——当今NBA换防能力顶级的中锋，挡在了穆雷与篮筐之间。穆雷突到底线将球抛出，球砸在篮板上沿，在篮筐上颠了两下穿网而入。分差来到17分，比赛彻底失去悬念，掘金拿到了2-1的总比分，他们距离总冠军的目标还差两场胜利了。穆雷交出34分10篮板10助攻的数据，他成为NBA历史上第3位在总决赛打出得分30+三双的后卫，之前两位是1969年的杰里-韦斯特与2017年的库里。这场球是穆雷复出后参加的第83场比赛，距离膝盖受伤过去了786天，他从重伤的深渊攀登了上来，站上了比受伤前更高的山峰，向荣誉之巅进发。贾马尔-穆雷的父亲，在9岁那年跟随家人从牙买加来到加拿大。年轻时的罗杰打过篮球，练过武术，是李小龙的铁粉。在穆雷还很小的时候，罗杰就给他放李小龙的电影。罗杰认为功夫的精髓不仅是强身健体，还有意志力的锻造，打篮球也应该如此。穆雷从3岁开始练球，父亲给他设计了很特别的训练计划，当室外风雪交加，别的孩子都躲在屋内，正是父子俩特训的好时机。穆雷在湿滑的场地上练运球，冒着雪做俯卧撑和深蹲。在穆雷进行深蹲练习时，罗杰会在他的腿上放一杯热茶，检验动作是否标准。罗杰在投篮训练上也是“没困难创造困难”，罚球时必须连续罚进30个，丢一个就要从零再来，这样可以锻炼技术稳定性和心理抗压力。罗杰还让穆雷在常规投篮训练完成后，加练闭眼投篮，这是球感的培养，为应对高强度的防守而准备。“父亲对我是军事化管理，他在建立并挑战我的信心，”穆雷说，“我从小在冰天雪地中打球，看不清篮筐，手冻僵了......这些对我都没有影响，当我在正规的球馆里比赛，还有什么会干扰到我吗？”从小的磨练让穆雷的篮球生涯很早熟，6岁的时候就加入10岁年龄段的小联盟，12岁开始和高中球员打比赛，18岁代表加拿大出征泛美运动会，半决赛打美国队第四节和加时赛贡献22分带队取胜。NBA传奇控卫当时担任加拿大队总经理，他为穆雷的比赛感到惊讶。“他很特别，才18岁就能从容应对这样的大场面。”纳什说。纳什在国家队还见识到了穆雷的训练态度，他给自己设定了加练计划，在球队常规训练结束后要继续练两个小时，教练怕他这么练会伤了身体，屡次劝说未果后，把他的球鞋藏了起来，但结果他们发现穆雷光着脚练了两个小时。穆雷将他玩命练的精神带到了NBA，成为了球员圈子内著名的“球馆老鼠”，这种职业素养与产生了共鸣。2019年的夏天，穆雷参加了科比创办的曼巴训练营，这个篮球营是邀请制，当时与穆雷一起入营的还有莱昂纳德、乔治和欧文等球星。科比带着穆雷一起观看比赛录像，进行身体和技术特训。科比专门传授了穆雷利用下肢力量进行单打的技巧细节，令穆雷受益匪浅。在训练营结束时，科比还和穆雷相约下次再见，但仅仅五个月后，科比就遭遇直升机事故离开。“那次训练营，是我第一次，也是最后一次和科比练球，我学到了许多，我希望能够将他教给我的带到比赛中，这是我怀念他的方式。”穆雷说。穆雷兑现了他的承诺，在2020年季后赛给出了场均26.5分，两场50+的表现，帮助掘金杀入西决，当时穆雷才23岁，似锦前程就摆在他的眼前，但仅仅半年之后，篮球生涯最大的打击就到来了。“忍受疼痛是篮球运动员应该做的，你要做一个硬汉。”2020年2月11日，掘金对马刺一战半场休息时，罗杰走进更衣室，望着正在缠绷带的穆雷这样说道。这场球穆雷刚从脚踝伤势中恢复过来不久，他在上半场又扭到了脚，导致两节比赛5投1中只得到3分。罗杰认为穆雷是心理上出现了胆怯，这可不是穆雷家族的赛场品格，他要鼓励儿子战胜疼痛扭转比赛局面。罗杰的心灵鸡汤果然有效，穆雷在下半场16投10中杀下23分，带领掘金击败了马刺。穆雷在赛后拥抱了父亲，他为穆雷家族的坚韧感到骄傲。带伤坚持对于穆雷是习惯操作，他渴望成为赛场硬汉，不因伤痛而退缩，但这样的强悍，也埋下了隐患，并最终以穆雷最不想看到的方式爆发出来。2021年4月13日，穆雷永远不会忘记的一天。掘金在客场对垒勇士，穆雷之前因为右膝疼痛连续休战四场，球队原本计划让穆雷再歇几场，但他主动请缨复出。在比赛还剩50秒时，穆雷运球突破左腿在触地的一刹那出现了很不自然的扭曲，穆雷摔出了底线，一只手抓着左膝，另一只手拍打着地板。掘金主教练迈克-冲到穆雷身边，当他看到穆雷痛苦的表情就意识到情况不妙，立即呼喊医务人员上场。队医推出了轮椅，但穆雷摆了摆手拒绝，他坚持在搀扶下走回更衣室，左腿已经无法用力。穆雷和掘金起初期待只是一次膝盖过度伸展，但检查结果令他们失望了，穆雷左膝十字韧带撕裂无限期休战。即便是运动医学已十分发达的今天，十字韧带撕裂也是可能摧毁运动生涯的重伤，养伤通常需要一年，并且爆发力恐将出现不可逆的衰退。当穆雷得知自己的伤情时，他的情绪崩溃了。“那种伤对于职业球员可能是毁灭性的，他在受伤后第二天找到我，哭着问我球队会不会放弃他，我告诉他不会的，他是我们的球员。”马龙回忆道。手术后的恢复是漫长并且艰苦的过程，由于长时间无法正常活动，穆雷的左腿出现了严重的肌肉萎缩，连最普通的伸展训练都不能完成。穆雷如同小孩儿学步一样从零做起，先是慢慢地走路，然后逐步上强度，当伤后第一次做深蹲练习，这个他从小做过无数次的项目，穆雷试了好久才比较规范地完成了一次。当进度来到弹跳训练时，先是跳台阶，然后是跳箱子，曾经无所畏惧的穆雷总是在脑中回想受伤时的场景，他害怕那一幕会重演。是父亲为穆雷打开了心结，罗杰编辑了2020年季后赛的集锦放给穆雷看，那些荡气回肠的比赛，唤醒了穆雷心中被重伤压制的斗志。“爸爸带着我看录像，一起研究哪里做得好，哪里还需要改进。我看到自己受伤前的样子，这令我坚定了目标，无论付出多么大的辛苦，都要杀回去。爸爸说心理关要比身体关更难过，只有我忘掉膝盖曾受过伤，我才是真正的康复了。”跨过心理阴影的穆雷，花了整整一年半的时间，终于回到了赛场上。2022年10月20日掘金客场对爵士，穆雷复出后的常规赛首秀，他打了26分钟得到了12分。“我身上还有一些锈迹，但我回来了，从这里出发，继续向前。”穆雷说。约基奇在三分线外做掩护，穆雷运球切入带走防守人回传给约基奇，创造出一个空位投篮的良机，约基奇出手但未能命中。随着掘金进入2022-23赛季总决赛，穆雷与约基奇的这段配合视频在网络上传播，这一幕发生在9年前，穆雷和约基奇参加在波特兰举行的篮球峰会，共同代表世界联队迎战美国队。那届比赛的明星并不是穆雷与约基奇，唐斯和贾利尔-奥卡福才是，当时约基奇还没有决定来NBA打球，穆雷还没上大学，再厉害的预言大师也不会想到他们有朝一日会成为NBA最令对手头疼的双子星，但穆雷回忆起当年的那次合作时，会有种子已经埋下等待生根发芽的感受。“我们在球场上拥有相同的篮球语言，都会去阅读比赛，相互创造机会，那种默契很难用言语描述。”2015年，约基奇来到掘金，一年后穆雷加盟。合作初期两人都打过一段时间替补，他们在那段日子一起研究战术，相互了解比赛习惯，培养出了不同寻常的篮球化学反应，这源自他们各自篮球技能的多样性。穆雷和约基奇都是投篮覆盖半场的无死角投手，自带精英级传球技能包的约基奇可以作为控卫使用，而穆雷除了可以在组织者和终结者之间切换战术角色外，还拥有挡拆掩护功能。与湖人的西部决赛第三场战至末节关键时刻，就是穆雷做掩护带走了安东尼-戴维斯，给约基奇创造出三分机会一击致命，这个战术走位并不是马龙教练安排的，而是穆雷与约基奇在场上根据防守布阵随机应变的成果。“当时他们坐在板凳上说同一侧站位，一定能正确阅读防守给出回应，我相信他们没问题，比赛的进展也正是如此。”马龙说，“有些人不愿意做掩护，因为很费力，很多时候也拿不到数据，但贾马尔不是这样，无论对方是大个子还是小个子，他都愿意去扛，因为他知道自己做掩护，让约基奇去持球，他们可以打出多变的战术，让对手防不胜防。”穆雷在2023年季后赛场均27分，比三年前那波季后赛状态大爆发时还要好，他能够在经历大伤后恢复战力，甚至比受伤前更强，除了自身的努力外，与约基奇的协作也是驱动力。两人之间的配合，可以将比赛变得简单。总决赛第二场，约基奇拿到了41分，但掘金被热火击败，症结在于热火割裂了约基奇与队友尤其是穆雷之间的连线，将约基奇放在了“孤岛”上。第三场比赛穆雷与约基奇共同做出调整，穆雷主导了更多的进攻回合，约基奇为穆雷做了32次掩护，这个数字是他们从2016年并肩作战以来第二多，穆雷利用掩护或者攻筐或者吸引防守回传约基奇，约基奇拿到了32分21篮板10助攻，穆雷34分10篮板10助攻，NBA历史将常规赛和季后赛都算上，这是第一次出现队友同场比赛都拿到得分30+三双的情况。“这关于信任和比赛感觉。如果他把球给我，他知道我能做什么。我们已经不需要讲太多，只需要根据场上的情况做出决定就可以，我们相互之间太了解了。”现在，掘金距离总冠军还差两场胜利，但没有人可以保证他们一定会抵达目的地，热火一路黑八上来，反复证明轻视他们是要付出代价的，两场胜利看似不远，但越接近山顶的路越难走，唯有脚踏实地。“我相信他会做到的。”父亲永远是穆雷最坚定的支持者，无论穆雷在高峰还是在低谷，“我知道他为梦想付出了多少汗水，他不会放弃，也不会让任何人阻挡他向目标迈进的脚步。”老胡聊球
