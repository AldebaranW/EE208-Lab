独家专访｜大哥阿联，“躺平”一年|易建联|奥运会|杜锋|科比|姚明_网易体育
2021-06-07
独家专访｜大哥阿联，“躺平”一年用微信扫码二维码分享至好友和朋友圈5月的羊城已经入夏，即便是在室内，的手里都需要攥着几张纸巾，时不时擦着汗。网易体育抵达现场时，他正在薪火阵营合伙人的引导下参观新校区，从网球场、舞蹈教室再到恒温泳池，每一处教学场地都没有落下。大部分时候他只是静静地站在一旁倾听或是随口搭上几句话，直到走进篮球场，他的问题多了起来。StartFragment易建联对未来有过各种规划，继续办好薪火训练营便是其中之一。但因为计划总赶不上变化，他说唯一能确定的，就是自己离不开篮球，“反正还是在圈子里，因为自己喜欢这个，也是做这个的。大家都说一个体育人嘛，一辈子都不会离开体育。”开始结束1像曼巴一样“大家好久不见。”3月CBA全明星技巧挑战赛前，易建联穿着高领针织衫和西装来开球，从场边起身走向特邀嘉宾吉克隽逸的那几步路，很短也很漫长，所有人的焦点都在他七个月前进行过跟腱手术的右脚上。易建联面带微笑向全场问好，语气轻松得仿佛离开球场的这段时间只是他给自己放的一个长假，如今假期过半，回来给家人和朋友们报个平安。曾经这样形容易建联——“只要他站在那里，就给所有人打了一针强心剂。”亮相全明星周末的易建联虽然无法上场比赛，但他还在，就已足够。简简单单的六个字，触发了全场最热烈的掌声，更有球迷在他开口的瞬间潸然泪下。上一次他们这样流泪，是在2020年CBA总决赛第三场。第三节剩下不到一分钟，易建联在罚球不进后起跳抢篮板，在落地的那一刻，他先是用左手轻撑了一下地面，随后用右手牢牢地抓住了右脚脚踝。他在地上躺了很久，表情从最初的痛苦到趋于平静，最后甚至有一些认命的意味。整个过程中，从对手到队友，没有人和他发生过身体接触。跟腱断裂就是如此，不需要多么激烈的对抗，只像是有人在后方轻轻踢了一脚，那是疲惫的身体在用相当温柔却又极其残忍的方式告诉你，它撑不住了。StartFragmentEndFragment19/20赛季CBA总决赛第三场，易建联跟腱受伤倒地比赛还没有结束，但在当下，易建联的伤情甚至超过了大家对胜负的关注。跟腱受伤对于一名篮球运动员的打击是毁灭性的，即便能够归来，也难复当年勇。所有人都害怕但又不得不抛出这样的疑问：易建联会不会退役，世界杯难道就是国家队11号的绝唱吗？万圣伟和杜润旺原本准备将易建联抬回更衣室，但他还是选择扶着队友，单脚走了出去。离场前，易建联看了一眼比分牌，91-84，广东领先。他说那是当时自己最关心的。曾被问过是怎么做到跟腱断裂后还能够罚中两球的，回答是“当有事情比疼痛更重要的时候，你就会忘了。”在8月15日的青岛钻石体育馆里，易建联的心里也有比疼痛更重要的事。在周鹏眼里，易建联太不容易了。“他没有跟任何人说，我最近膝盖怎么了，我最近拉伤了会影响得分，他没有给自己找很多借口，只是默默地付出，默默地为我们承担。”谈起易建联时周鹏在不停叹气，是一种 “为什么伤病偏偏降临到我兄弟身上”的惋惜。同为球员，他太知道在职业生涯这个时期受伤意味着什么；身为队友，他更知道易建联走到如今这一步，承受了多少。左膝盖严重积水肿胀、跟腱老伤、右腿肌肉拉伤，这些都不是易建联在接受采访时的高频词汇；愤怒、沮丧、无奈，也不是你能轻易在他脸上捕捉到的情绪。镜头记录下的易建联，会淡然地看着工作人员给自己固定受伤的右脚，似乎那只是一个再正常不过的赛后流程，然后被苏伟背着，去和队友一同捧起第十座联赛冠军奖杯。StartFragmentEndFragment8月26日，易建联在社交媒体发布了名为“我们都不会变成另一个科比，但我们都可以用自己的方式来实现‘坚韧’”的长文。他告诉大家手术很顺利，不必为他担心。他说伤病是运动员必须面对的坎，他已经学会怎么与它们相处并且战胜它们。易建联以科比为题，更像是在鼓励自己，曾经有球员和他一样经历跟腱断裂，花了八个月恢复，继续打了两年，还能在退役战砍下60分。那一场比赛，易建联也在现场。2竞技体育就是挑战“一个人把好胜坚持了几十年，上千场比赛，那些星星点点的光亮，最后汇成了一道银河。”易建联把他在洛杉矶看到的银河，带到了2019年的广州。男篮世界杯中国对阵尼日利亚，在一场争夺入场券的比赛中，易建联拿下全场最高的27分却依旧无力回天。距离比赛结束还剩不到50秒，易建联被换下场，走到替补席拍了拍他的肩膀。整个广州体育馆响彻易建联的名字，这样的呼喊声在比赛终了双方球员握手时再次爆发。接受外媒采访时，流着汗的易建联用最平静的语气说出最狠的话：“I try to play it as my last game of the career，to see if we can get a win.（我把它当成职业生涯最后一场来打，以期能够取得胜利）”2019年男篮世界杯，“老兵”易建联展现了队长担当2胜3负，抽到上上签的中国男篮在主场被打得灰头土脸。在媒体和球迷的口诛笔伐中，易建联的名字却成了一片禁区。当#心疼易建联#的话题冲上热搜榜时，年轻一点的球迷或许很难想象，这是一个当年被嫌弃无法接过姚明在国家队、NBA大旗而被群嘲的球员。互联网是有记忆的，时至今日，当你在搜索引擎敲下“N投一中”时，关于“太空易”的各种段子还会弹窗而出。而按照时下热门的说法，易建联是凭着业务水平口碑逆袭了。“魔鬼筋肉人”是在易建联身上最常见的标签之一，甚至不需要多了解体育的人都知道，想要十年如一日维持这样的肌肉形态有多难。可在刚接触篮球那会儿，别提肌肉了，跑步对他来说都是天大的难事。“我没想到打篮球是要跑圈的，而且一跑就是3200米，整整绕着操场跑八圈！那时候我的极限是300米，想想那些日子都觉得腿酸。”好不容易跟上了广东的训练强度，易建联又在国家队接受了一次震撼教育，3200米升级到10000米，整整25圈！“进国家队那一年我是队里最年轻的，队里有姚明、李楠、巴特尔，应该还有郭士强、张劲松。那时候我想，作为一个年轻球员，我的体力应该算很好吧，但还是感觉到整个国家队的训练量非常大。他们几个老球员身上其实都有很多伤病，但他们都在用自己的行动来表态，去全面训练，为了奥运会不断付出，希望能够为国家队取得一个比较好的成绩。从那时候就能感受到老一辈球员传承下来的精神，我学习到了，希望变成自己的东西。”王仕鹏就对易建联后来的魔鬼训练习惯印象深刻。他在一次采访中回忆，国家队在宁波工体集训时，从上午到下午全是训练课，当所有人都练不动的时候，只有易建联还会在晚上去加练。每天易建联都会最早抵达训练场练习投篮，等队友来了，他已经练完一轮了。这些血汗日子，到了如今的易建联口中，都变成了再平常不过的事：“既然打篮球是一种职业，你必须要养成一种习惯。”易建联总会最早到达球馆训练2020年9月，国家男篮集训队在国家体育总局训练局举行入队仪式。没有到场的易建联给后辈们写了一封信，没有太多华丽的辞藻，他只是在叮嘱大家：“国家队是让你们玩命的地方，是让你付出一切帮助全国球迷完成梦想的地方。前辈们用血汗奋斗出佳绩，希望你们也一样去努力、奋斗、继承。”如果你对21岁的易建联在北京奥运会对阵德国队时的准绝杀还有印象，看过25岁的易建联在伦敦奥运会面对伊巴卡和加索尔的组合得到30分12板，你就会知道33岁的易建联写下的玩命，是一个真真切切的动词。一个月前，国家队公布了备战奥运会落选赛和亚洲杯的集训名单，易建联的名字赫然在列。有球迷调侃“大哥可能是想蹭国家队的空调了”，有人推测国家队不过是想拿他当梦八队的基德，家有一老如有一宝。总而言之，没有人当真。可到了这个月初，当摄影师拍到易建联穿着被汗浸湿的衣服走出训练馆，当央视的镜头拍下他已经开始参加对抗训练时，你就会明白他所说的，“只要有机会能够进入国家队，对我来说都是很值得期待的，是我继续努力的目标。作为一个运动员，（入选国家队）是至高的荣誉。”大伤过后，易建联出现在最新的国家队集训名单StartFragmentEndFragment明眼人都知道，以中国男篮现阶段的整体实力，哪怕是拥有一个巅峰期的易建联，落选赛小组突围的可能性都微乎其微；而跟腱重伤后贸然上场的代价，2019年的杜兰特就是现成的反面教材。易建联怎么会不知道挑战很大，可在解答记者的困惑时，他依然没有搬出伤病当挡箭牌，而是给出了虽千万人吾往矣的态度：“体育运动就是一种挑战，所有比赛对我们来说都是越挑战，就越期待。目前的身体状况一切都还可以，都挺顺利的。”3旗帜的力量即便已经在职业赛场摸爬滚打近二十年，参加过数不清的媒体采访，但易建联依旧不习惯在镜头前谈论自己，更不要说是接受他人的称赞。天才、专注、内敛和成熟，当被网易体育问到哪一个词最贴近自己时，易建联低着头听着，嘀咕了一句：“我从来不会用这些词来形容自己。”偏偏，这些是广东队主教练杜锋在谈到易建联时，最喜欢用到的几个词，可能还要加上一个“领袖”。今年的CBA总决赛易建联很清闲，这是他第一次坐在场边，看完了系列赛。广东队拿下第十一冠后，球员通道边的球迷开玩笑地朝他喊了一句“大哥躺赢啊”，心情不错的易建联和他击了掌。但易建联同时又很忙碌，他被杜锋拉去接受了没有任何准备的场边采访，不善言辞的他还被要求和队友多说几句。“他是我见过最努力的运动员之一，特别努力，”杜锋第一次见易建联还要追溯到2002年，他也许不会想到当年国青队里的青涩少年，如今是自己执教球队中的精神支柱，“阿联来球队其实就是给年轻球员鼓舞和支持。”总决赛第三场生死战前，杜锋让易建联给队友打打气，他甚至感受到，如果当时自己派他上场，易建联也不会推辞。“他肯定没有完全康复好，但我能感受到他对篮球的热爱和渴望，对这个冠军的渴望。”在外人看来，这可能只是杜锋的美好愿景，乃至威慑辽宁队的一种策略，但当网易体育向易建联求证时，他扭头笑了笑，下意识地回了一句“会吧”。“也没有，”玩笑过后，那个理性的易建联又往回找补了一句，“坐在场下看比赛，任何时候都是希望能上场比赛的。”易建联说看着场上的胡明轩、赵睿，总是会想起自己初入广东队时的情况，他知道留给年轻队员适应的时间很短，需要马上进入角色，贡献自己的力量，能看到他们在不被看好的情况下拼下冠军，非常欣慰。就在几年前，胡明轩还是训练营里找易建联签名的小迷弟，如今儿时偶像变队友，甚至能在大哥因伤缺席的时候担起MVP的责任，对他来说是一件特别不可思议的事。“小时候看到联哥对于中国篮球的影响力，包括他在NBA的一些经历，就非常崇拜。”都说近距离接触偶像容易幻灭，但胡明轩完全没有这样的困扰：“特别幸运自己能和联哥成为队友，更加可以看到他对待平时训练、比赛的专注度。他是我们所有人的一个榜样，在队里面绝对是一个旗帜。他训练比我们年轻球员还早到，特别刻苦。”“大哥”易建联和新科FMVP胡明轩合影易建联说不清所谓传承具体要做些什么，但他知道想要奔跑的心终究会斗不过想要刹车的身体，所以他所能够做的，就是通过在训练里的一举一动，让后辈看到自己日复一日、年复一年对待比赛的态度；是在社交媒体上发出和胡明轩的合照，然后鼓励小朋友：“希望你不忘初心，继续努力。”一代人终将老去，但总有人正年轻。养伤的这段日子里，除了想着国家队和宏远，易建联也拥有了很多时间思考球场以外的事。易建联的薪火训练营规模逐渐扩张，他希望在目前的基础上做出高品质，这样总有一天能够挖掘到特别好的体育苗子，为国家争得荣誉。他的两个孩子也在长大，开始在学习之余接触各项体育运动，易建联希望他们会希望喜欢篮球，不需要刻意取得什么成绩，能成为一种兴趣足矣。“体育这个概念在近十年左右，在人群生活中的占比越来越重，已经慢慢变成一个刚需。以家庭为单位来说，包括现在我们作为父母，自己做体育嘛，希望小孩能够德智体全面发展。我自己的父母，他们每天也在做自己力所能及的一些运动，去保障自己的身心健康。所以以家庭为单位或是到整个大环境来说，体育都是生活里面很重要的一部分。那么不管是疫情还是什么情况，体育最终还是鼓励人们走出来。”2016年，易建联出席薪火阵营发布会看着对体育产业发展侃侃而谈的易建联，让人隐约预见了他球衣褪去、西装革履的未来。所以在采访的最后，记者试探性地问出“老板易建联和球员易建联有什么不同？”那是他在整个谈话中反应最快的瞬间。“只有球员易建联”，他的回答不带一丝犹豫。StartFragmentEndFragment阿廖铲球
