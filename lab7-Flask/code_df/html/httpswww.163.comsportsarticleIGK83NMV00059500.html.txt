预防出生缺陷 科学健康孕育 彩票公益金为患儿点亮希望之灯用微信扫码二维码分享至好友和朋友圈我国是出生缺陷的高发国家之一，为加强出生缺陷综合防治，从源头保障优化生育政策实施，在财政部、国家卫生健康委支持下，2015年中国预防出生缺陷干预救助依托中央专项（以下简称“中彩金”）启动了全国出生缺陷干预救助项目，从生命起点关注民生。为生病患儿雪中送炭“有了彩票公益金的资助，缓解了我们家庭的经济困难，非常感谢中央专项彩票公益金出生缺陷干预救助项目的援助。”芃芃（化名）的爸爸感激地说。芃芃出生时就患有先天性疾病尿道口下裂，当时医生建议手术治疗，然而由于家庭比较困难，只能暂缓治疗。在芃芃2岁时，他的父母凑足了1万元钱，带他进行了手术。然而第一次手术效果却并不理想，芃芃术后经常漏尿。“家里买不起尿不湿，孩子第一次治疗效果不理想，我们就用旧衣服剪来当尿布缓解，直到凑齐了第二次治疗的费用，我们特意挂了专家号重新确定治疗方案。”芃芃的爸爸表示，“也是这时得知有中央专项彩票公益金支持的出生缺陷干预救助项目可以为我们提供帮助。”。在第二次治疗过程中，芃芃的爸爸在武汉市儿童医院遇到了王军主任医生，得知他家的经济情况后，王医生帮助芃芃申请了中彩金出生缺陷干预救助项目先天性结构畸形救助。幸运的是，第二次手术很成功，在芃芃出院1个月后，他们收到了中国出生缺陷干预救助基金会发放的1万元救助金。这笔钱对于芃芃一家来说意义重大，缓解了芃芃一家因为治疗费用带来的债务压力。为困难家庭减轻负担“算下来，这次给孩子手术我们几乎没有花多少钱，真的是感谢国家政策、感谢中央专项彩票公益金支持的出生缺陷干预救助项目给我们的帮助，没想到已经7岁的孩子依然可以享受到这份福利。”优优的爸爸很是感慨，他根本没想到自己的孩子能够享受到这一政策的帮扶。优优出生在陕西省咸阳市永寿县一个普通的农村家庭，全家依靠务农为生，原本平静的生活在优优3岁后逐渐发生改变。“优优3岁前偶尔尿床，因为孩子还小，我们也没有特别在意。但是3岁后，我们发现优优尿床频率非常高，需要经常叫醒他排尿，而且需要给他铺很多尿布，这种状况一直持续到优优7岁。”优优的爸爸介绍道。眼看优优的症状越来越严重，今年5月，优优的父母带着他前往西安市儿童医院做了检查。经过一系列检查，医院专家诊断优优患有先天性脊髓栓系综合征。这一结果对于优优一家来说简直是晴天霹雳，明明只是小孩子常见的尿床，竟然被确诊了先天性疾病。“家里两个孩子都是靠我们务农收入，看天吃饭的日子经不起任何打击。”优优的家人在得知优优需要进行手术时陷入了无助，既担心孩子手术有风险，又发愁手术费用如何凑齐。了解到优优的家庭情况后，西安儿童医院的医生第一时间告知优优家人，优优的这种情况符合先天性结构畸形救助项目救助条件，可以申请中彩金出生缺陷干预救助项目的帮扶。最终，优优在手术后获得了该项目拨付的1.56万元救助金，极大地减轻了家里的就医负担。守护人生健康第一关芃芃和优优的故事只是中彩金支持出生缺陷救助项目的一个缩影。自2015年起，中彩金出生缺陷干预救助项目就启动出生缺陷救助、检测、防治宣传和健康教育等工作。截至2022年底，以体彩公益金为重要组成部分的中央专项彩票公益金累计支出15.46亿元资金用于支持出生缺陷干预救助项目。近年来，国家卫生健康委与相关部门积极推进出生缺陷综合防治，建立起覆盖全链条的出生缺陷三级防治体系。一级预防是在孕前的综合干预，通过遗传咨询、健康教育、预防感染等减少出生缺陷的发生；二级预防是通过孕期筛查和产前诊断识别胎儿的严重先天缺陷，早期发现、早期诊断减少缺陷患儿的出生。三级预防，即出生后早筛查、早干预，能够尽早康复，尽最大限度减少出生缺陷所导致的残疾。中彩金出生缺陷干预救助项目的开展主要围绕三类疾病进行救助，包括：遗传代谢病、先天性结构畸形病、功能性出生缺陷病。其中，遗传代谢病救助主要包括对可治、可控的遗传代谢病患儿实施救助，减少致残、致死等严重后果的发生；先天性结构畸形疾病救助范围包括7大类74种先天性结构畸形疾病；功能性出生缺陷救助为“十四五”期间新增救助范畴，病种覆盖八大类56种功能性出生缺陷疾病。据悉，患有上述三类疾病的18岁以下家庭困难且患有出生缺陷疾病的儿童，医疗费用自付部分超过3000元的，都可以提交材料，审核通过后可以申请到最高3万元的相应比例的救助，诊断、治疗、手术、包括康复等费用都在救助的范畴。据中国出生缺陷干预救助基金会工作人员介绍，截至今年8月31日，中彩金出生缺陷干预救助项目共完成新生儿检测263万多例，救助出生缺陷患儿57378人次，为孕期妇女提供27262人次产前筛查和1650人次产前诊断服务。在中彩金持续对出生缺陷干预救助项目的支持下，我国出生缺陷综合防治服务的均等化水平已经明显提升。国家卫生健康委发布的最新数据显示，2021年与2011年相比，出生缺陷导致的婴儿死亡率、5岁以下儿童死亡率分别从2.3‰、2.7‰降至1.1‰和1.2‰，下降幅度均超过50%。遗传代谢病的筛查干预率不断提高，极大降低了患儿智力神经发育滞后的概率和不良影响，有效提升了患儿的生存质量。中彩金出生缺陷干预救助项目为推进健康中国建设，完善出生缺陷综合防治服务，有效减少因病致贫返贫作出重要贡献。（试翼）法眼儒生
