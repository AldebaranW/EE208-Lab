广厦15分胜天津完成横扫 胡金秋24+10李京龙20分|天津队|广厦队|布拉彻_网易体育

广厦15分胜天津完成横扫 胡金秋24+10李京龙20分用微信扫码二维码分享至好友和朋友圈网易体育1月31日报道：时间1月31日晚，常规赛第38轮主场迎战队。实力占优的队全场比赛掌控场上的主动权，最终他们115-100大胜天津队，本赛季常规赛“四杀”对手完成了横扫。四节比分分别为：23-27、26-25、32-36、19-27（天津队比分在前）。技术统计：天津队：罗切斯特27分6助攻，27分15个篮板3盖帽，时德帅20分；广厦队：24分10篮板，福特森17分12次助攻8篮板，20分，博洛西斯18分9篮板，林志杰12分，刘铮12分。赛事相关：广厦队目前以27胜9负排名积分榜第4位，天津队仅取得9胜26负只排名第17位。本赛季常规赛前3次交锋，广厦队全部获胜。比赛回放：福特森突破上篮打进，揭开了本场比赛的序幕。孟子凯篮下打成2+1，刘铮与布拉彻先后内线得分，罗切斯特急停三分命中，天津8-4领先广厦。胡金秋连取4分追平比分，刘铮快下反击实现反超。天津队外线屡投不中，被广厦连续打成快攻拉开6分分差。布拉彻、李京龙、罗切斯特相继飙中三分球，广厦22-17天津。罗切斯特与福特森展开对攻取分。首节结束，广厦27-23领先天津。第二节比赛，双方国内球员相互取分。李京龙连得5分后，广厦34-27领先7分。布拉彻以一敌三篮下打进，接下来又强行冲击内线造杀伤，2罚全中将分差缩小到2分。博洛西斯勾手命中，林志杰断球反击打进。罗切斯特连取4分止血，林志杰立即回应三分远投。博洛西斯2+1打成，罗切斯特三分命中，继续紧咬比分。胡金秋单臂扣篮，罗切斯强轰三分迫近到1分。胡金秋低位进攻再添2分，罗切斯特造犯规2罚全进，博洛西斯压哨勾手命中。半场战罢，广厦52-49领先天津。易边再战，广厦队提升防守强度，胡金秋、林志杰连续反击打成，一波8-2攻势将分差拉大至9分。天津队追回4分，博洛西斯三分飙中，胡金秋空切打进，金鑫空位三分中为天津止血。之后两队交替得分，广厦保持着15分左右的领先优势。之后广厦队有所松懈，罗切斯特、时德帅反击打成追到9分差距。李京龙三分命中稳定局面，天津队进攻提速继续咬住分差。随着时德帅最后一攻三分命中，三节结束，广厦88-81领先天津。最后一节，福特森的跳投和李京龙的三分，令广厦领先到了12分。时德帅远投轰进，胡金秋、刘铮相继建功，李京龙底角三分轰进，广厦100-86打停天津。福特森远投再中，分差拉大至18分。战至本节后半段，牢牢掌控优势局面的广厦开始走马换将，锻炼年轻球员。最终广厦115-100大胜天津，常规赛“四杀”天津完成了赛季横扫。双方首发：天津：布拉彻、田野、时德帅、孟子凯、罗切斯特广厦：赵天熠、林志杰、胡金秋、福特森、刘铮篮球资讯达人
