中国男篮主帅：球队目标不设上限 努力去赢得奖牌用微信扫码二维码分享至好友和朋友圈新华社12月26日电（记者曹奕博）中国男篮新任主教练塞尔维亚人亚历山大·26日在接受新华社记者专访时表示，对中国男篮的未来有很高的期待，国家队队员选拔将考察球员的全面素质与团队贡献能力。自上月18日抵达中国后，男篮新帅乔尔杰维奇就马不停蹄地奔赴CBA赛场，开启教练团队组建和国家队队员考察工作。乔尔杰维奇表示，一个多月的考察还不能对中国男篮给出精确的评价，但他在担任塞尔维亚主教练期间对中国篮球已经积累了一定的了解。乔尔杰维奇表示，CBA联赛有很强的比赛强度和身体对抗，许多有天赋的年轻球员给他留下了深刻的印象。新主帅在场边看球时给球员“评星”、写备注的视频近期受到了外界的较大关注，对此乔帅表示，用一至三颗星打分只是他记住球员的小习惯，正式选拔时将考虑到球员的各方面能力。“首先球员要有很好的职业素养和荣誉感，披上国家队战袍是一个球员能取得的最高荣誉。”乔尔杰维奇说，“除此之外，球员需要具备很强的求胜欲和出色的球商。在我的篮球理念里，‘精英’球员要具有对比赛和自身位置的解读能力，他们不光要明白能为俱乐部带来什么，更要清楚自己在国家队中扮演的角色。”乔尔杰维奇认为，构建国家队不是选最好的12名球员，而是选最合适的12个人，将团队优势最大化。“我的最终目标是组织起一个强大的整体，每个球员都要无私地传导球，我看中的是团队的胜利而不是个人数据。”他说。谈及这份新工作，乔尔杰维奇表示，中国篮协给了他很大的支持。“我和篮协主席在一起花了很多时间，他帮助我更好地了解中国篮球体系和中国球员。姚明在国际篮球界的公信力促使我痛快地接受了这份工作，他的思路和眼界给我的工作提供了宝贵的帮助。”2023年8月底开赛的男篮将是对中国队的一次大考验，乔尔杰维奇执教的中国男篮将在世界杯上寻求突破以争取一张2024年巴黎门票对于国家队下一步工作，乔尔杰维奇表示体能训练是重中之重，“在现代篮球发展的趋势中，身体强度在比赛中的重要性已经超过了技术能力。”乔尔杰维奇表示，与他合作超过十年的体能教练米哈伊洛维奇将带来团队和训练体系。同时，乔尔杰维奇也强调了多与强队交手的重要性，他计划与篮协进行沟通，尽量在中国男篮备战世界杯期间安排足够的热身赛。“要战胜一支球队就要多与之打比赛，这样才能适应他们的比赛节奏，同时根据他们的特点找出突破口。”乔尔杰维奇说。直播吧 2023-11-01 21:56:37李广专业体育评论 2023-11-01 21:37:22富贵少女日记 2023-10-31 09:38:42后厂村体工队 2023-11-01 09:35:41东球弟 2023-11-01 22:29:30体坛知识分子 2023-11-01 06:05:03篮球实战宝典 2023-11-01 06:00:03侃球熊弟 2023-11-01 09:37:239点体育 2023-11-01 18:09:22德融视讯 2023-11-01 13:52:10体坛故事集 2023-11-01 12:21:51青岛文艺 2023-11-01 14:55:51钉钉陌上花开 2023-11-01 23:29:31中国新闻周刊 2023-11-01 13:38:46湘楚风云 2023-11-01 22:02:41懂球帝 2023-11-01 05:37:33财联社 2023-11-01 09:55:27闻史 2023-11-01 09:43:20东方豪侠 2023-11-01 08:11:10东球弟 2023-10-30 12:50:05网易体育 2023-11-01 12:36:21东球弟 2023-11-01 08:52:34澎湃新闻 2023-11-01 20:27:13财联社 2023-11-01 15:20:38网易体育 2023-11-01 21:48:58央视新闻客户端 2023-11-01 15:59:20静海融媒 2023-11-01 14:57:48网易体育 2023-11-01 21:37:10星晨视频 2023-11-01 15:02:49澎湃新闻 2023-11-01 16:06:54滑县市场监管 2023-11-01 20:50:41中国海警 2023-11-01 13:17:44财联社 2023-10-31 22:10:49财联社 2023-11-01 15:43:45央视新闻客户端 2023-11-01 13:27:42不掉线电波 2023-11-01 14:29:21新京报 2023-11-01 18:14:13直播吧 2023-11-01 19:20:20不掉线电波 2023-11-01 21:44:48
