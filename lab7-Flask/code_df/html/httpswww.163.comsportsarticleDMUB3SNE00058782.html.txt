传统与现代的融合!首届户外那达慕于内蒙古盛装开幕用微信扫码二维码分享至好友和朋友圈网易体育7月17日报道：2018年7月13日，来自全国各地的近千名户外爱好者在自治区市观音山风景区一同见证了一场与众不同，别开生面的“CORP户外”。此次“CORP户外那达慕”是在传统节庆活动“那达慕大会”中深度融合户外运动内容，为全国的户外爱好者量身打造的，集草原风光观光游玩、传统蒙古族那达慕文化展演、蒙古族游牧文化体验、蒙古族美食工艺展出及品鉴、经典户外赛事体验、户外活动项目体验、户外电影节、草原文化节等，为一体的独立品牌IP的精品户外体育旅游项目。“CORP户外那达慕”首期活动于7月13日正式开始，将持续两周。据中国户外资源平台CORP发起人乔浩先生介绍，户外那达慕是为户外爱好者定身打造，可高度参与的草原户外运动和游乐项目。户外爱好者不仅可以体验内蒙古大草原的自然风貌和风土人情，更可近距离的在草原上进行露营、山地骑行、越野跑、皮划艇、滑翔伞、运动风筝、溜索、速降等户外项目，这是现代户外运动与传统民族体育的激情碰撞和深度融合，为古老的“那达慕”赋予了新的生命力。好客的蒙古族朋友以蒙古族迎客酒来接待来自五湖四海的户外领队。霍林郭勒市旅游局为当地户外旅游资源推介。领队们对当地的户外旅游资源和当地联合CORP中国户外资源平台共同打造CORP户外那达慕给予一致赞许。14日清晨在观音山上举行了神圣的蒙古族传统祭敖包仪式敖包祭祀活动结束后，当地牧民以游牧文化为主题进行了开幕式入场仪式。CORP户外那达慕开幕式于7月14日上午9:30分正式举行出席CORP户外那达慕开幕式的领导和嘉宾有霍林郭勒市人民政府副市长高金枝霍林郭勒市人民政府副市长杜盛林霍林郭勒市人大副主任徐凤莉霍林郭勒市委调研员门树贵霍林郭勒市教体局副局长孟根内蒙古自治区登山协会副主席柴勇CORP中国户外资源平台发起人、辽宁省登山户外徒步运动协会秘书长乔浩霍林郭勒市总工会主席宋继庆霍林郭勒团市委书记迟佳静霍林郭勒市委宣传部副部长杨景坤霍林郭勒市文广局副局长车瑜霍林郭勒市卫计局副局长万颖哲霍林郭勒市民宗局副局长包青龙霍林郭勒市旅游局副局长司结实霍林郭勒市文旅投公司董事长刘振东霍林郭勒市文旅投公司总经理陈雪峰霍林郭勒观音山风情园总经理贾永超CORP中国户外资源平台发起人辽宁省登山户外徒步运动协会秘书长
