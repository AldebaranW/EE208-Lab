男篮世界杯-立陶宛送美国首败 爱德华兹空砍35分用微信扫码二维码分享至好友和朋友圈网易体育9月3日报道：2023年男子篮球继续进行第二轮，队遭遇首败。虽然爱德华兹得到35分，但队7人得分上双，他们在首节确立优势后顶住对手反攻，立陶宛队以110-104战胜美国队。立陶宛队和美国队排在J组前两位晋级八强。四分之一决赛里，立陶宛队对阵塞尔维亚队，美国队迎战意大利队。立陶宛队的卡里尼亚斯卡斯得到15分和4个篮板，库兹明斯卡斯得到14分，瓦兰丘纳斯得到12分和7个篮板，布拉兹迪基斯得到11分和4次助攻，迪姆萨得到11分，塞德克斯基斯得到11分和11个篮板，本兹丘斯得到10分。美国队的爱德华兹得到35分，布里奇斯得到14分和3个篮板，布朗森得到14分和7次助攻，英格拉姆得到10分。开场的4平后赛德克斯基斯三分命中，瓦兰丘纳斯也投进三分，他们率队打出9-2点小高潮，立陶宛队拉开领跑。爱德华兹上篮还击，立陶宛队又得4分扩大优势。布里奇斯跳投暂时止血，卡里尼亚斯卡斯和迪姆萨各进三分，立陶宛队以23-10领先。里夫斯两罚全中，卡里尼亚斯卡斯又得6分，库兹明斯卡斯勾手命中，立陶宛队以31-12领先19分结束首节。本兹丘斯扣篮得手揭开第二节大幕，约翰逊三分命中，他率队连追7分，美国队以19-33落后。立陶宛队暂停后迪姆萨三分还击，英格拉姆后仰跳投命中，莫泰尤纳斯和库兹明斯卡斯各中三分，瓦兰丘纳斯两罚全中，立陶宛队重新领先15分。美国队之后又进攻失准，连续投篮打铁，立陶宛队又得4分扩大优势。凯斯勒上篮还击，瓦兰丘纳斯、布拉兹迪基斯扣篮得分，半场前2分50秒时立陶宛队以52-31领先。里夫斯、波蒂斯和班凯罗联手追回6分，斯尔维迪斯上篮得分，半场结束时立陶宛队以54-37领先。立陶宛队的瓦兰丘纳斯上半场得到11分和6个篮板，卡里尼亚斯卡斯得到9分；美国队的爱德华兹得到16分。美国队在第三节提升了防守强度，立陶宛队的命中率开始下降，布朗森和杰克逊各中三分，布里奇斯两次突破得手，他们带领球队打出15-2的反击高潮，美国队追至52-56。诺曼塔斯三分不中，美国队错失几次追分机会，莫泰尤纳斯勾手打进，塞德克斯基斯扣篮得手，立陶宛队以60-52领先。美国队请求暂停，布里奇斯传球失误，立陶宛队也出现失误，布里奇斯又得5分再次追近，库兹明斯卡斯上篮得手，三节结束时立陶宛队以71-65领先6分。立陶宛队连得4分开始第四节，他们再次领先10分。波蒂斯上篮回应，莫泰尤纳斯勾手打进，两队之后交替得分，立陶宛队牢牢保持领先，莫泰尤纳斯再次勾手得分，库兹明斯卡斯三分命中，第四节还有3分50秒时立陶宛队以94-85领先。英格拉姆和里夫斯错失三分，塞德克斯基斯两罚一中，双方分差回到10分。爱德华兹抢投三分命中，塞德克斯基斯四罚三中，立陶宛队以98-88领先10分。爱德华兹制造犯规两罚全中，迪姆萨又中三分，爱德华兹扣篮得手，杰克逊争抢时领到五犯毕业，约库拜蒂斯两罚全中，班凯罗上篮得手，爱德华兹再中三分，美国队追至97-103。库兹明斯卡斯两罚全中，布里奇斯三分中的，美国队还落后5分。诺曼塔斯被犯规两罚全中，立陶宛队以107-100领先。布朗森投三分制造犯规，三罚两中，双方仍是5分差距。美国队抢断不成只能犯规，卡里尼亚斯卡斯两罚一中，立陶宛队领先6分。布朗森罚中两球，美国队抢断成功，可惜三分不中，库兹明斯卡斯扣篮锁定胜局，立陶宛队最终以110-104获胜。美国队首发阵容：布朗森、爱德华兹、哈特、布里奇斯、杰克逊立陶宛队首发阵容：诺曼塔斯、约库拜蒂斯、塞德克斯基斯、布拉兹迪基斯、瓦兰丘纳斯影孖看世界
