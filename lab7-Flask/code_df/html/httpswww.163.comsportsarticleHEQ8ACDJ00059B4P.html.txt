Type="ordered" SerialNumber="01"@@“我真的从小到大一直在拿第一”北京冬奥会的赛场上，徐梦桃错过了混合团体的金牌，但是在随后进行的个人比赛中，她没有再次错失家门口登顶的机会。“我是第一吗？”这句话通过电视转播的镜头让每一个人都非常熟悉，徐梦桃的心声在这一刻动情迸发。“在我的体育生涯中，可能真的就是没有最后拿到第一，但是我真的一直在拿第一。”其实在登上雪道之前，徐梦桃在的赛场上奋战了很久。“从小练体育我就想拿冠军，所以我对这个词并不陌生，因为我刚练的时候，带过我的教练员都说，小桃子以后要拿奥运冠军！”家境并不优渥，徐梦桃希望能够通过自己的努力，让家里过上好日子。2002年8月份在辽宁省运动会的体操比赛中，当时还是一名体操运动员的徐梦桃拿到了3金2银1铜。荣誉之外，徐梦桃将自己所有的12000元奖金都给了父母。@@SectionTitle=" 成长的烦恼让徐梦桃加冕历史第一" Type="ordered" SerialNumber="02"@@每一个女孩，都会在成长过程中经历这个阶段——长高、发育，青涩的稚嫩逐渐绽放为青春的靓丽。不过对于运动员来说，这样的经历却是极大的烦恼，正是这个阶段，让徐梦桃从体操的赛场走向了未来的新时光。“正儿八经地控制体重都得有11年了。”面对记者的提问，徐梦桃还是会回想起每一次减重的苦恼。不过也正是这样，自由式滑雪空中技巧运动员徐梦桃才能够登上洁白的赛场。其实仅有17岁的徐梦桃就已经在国际舞台上崭露头角，她在世青赛中夺冠，这也是中国人首次站在世青赛的冠军领奖台上。2010年的温哥华冬奥会拿到了第六名之后，在索契的赛场上遗憾只收获了一枚银牌，2018年的平昌则是出现重大失误无缘领奖台……即便随着年龄的增大，徐梦桃在比赛的同时，还需要通过自己的传授来以老带新，但岁月沉淀之后更显从容。“其实在2018年平昌冬奥会之后，当时因为成绩并不好，加上回国之后还要进行手术，心理上会比较难过，会有一点委屈吧。”徐梦桃只浅浅谈及了自己的曾经的伤病。不过当我静下来的时候，我还是非常渴望走向北京冬奥会的赛场。在复盘比赛的过程中，当时我希望可以在北京遇见更强大的自己。”这位27次斩获自由式滑雪空中技巧世界杯个人金牌的“历史第一人”在家门口，向世界展示了中国冬季项目选手顽强攀越的运动精神。@@SectionTitle="健身让每一个彼此的距离迅速拉近" Type="ordered" SerialNumber="03"@@除了在自己挚爱的洁白赛场上取得突破，徐梦桃更是在人生的赛道上乘风破浪。2月14日冬奥摘金之后，与男友王心迪宣布恋情，进而在5月20日这个甜蜜的日子里，两人正式领证相伴一生，“妻子”的身份让徐梦桃的2022年更加圆满。同为自由式滑雪空中技巧运动员的二人，在日常的生活之中难免与运动密不可分，“很多训练我们都要在一起配合，包括上肢的训练、拉伸训练等等。我们两个比较喜欢去健身房，做一些核心训练和有氧间歇训练，来保持自己的运动习惯。”2022年5月20日当天，在沈阳领证现场，徐梦桃对王心迪说：“传承家风齐优秀，希望孩子未来可以继续传递正能量。”在将自己的运动基因传承给下一代之前，徐梦桃已经通过自己的实际行动身体力行的推广全民健身运动，让更多人都把健身运动融入到日常生活之中……在谈及自己之前参与综艺节目录制时，徐梦桃坦言，大家凑在一起的时候也会分享健身的心得。“桃桃，你平时要是练腹肌的话，会做哪些动作？”“平时的准备活动和运动后的牵拉要怎么做才更好呀？”“柔韧性要怎么练习呀？”……与身边不同的运动人士交流，让徐梦桃也进一步打开了自己的视野。在与记者交流中，徐梦桃表示：“其实每个人都是不一样的，有些柔韧性很好，有些协调性特别出色，综合能力也很强……其实大家在一起不管是唱歌还是跳舞，每天都过得非常有趣。”当然其中，健身的话题让彼此之间的距离迅速拉近。在中国体育彩票举办的社区健身云动会活动现场，专业的桃子姐对各类社区健身器材使用炉火纯青，连国家级健身教练都称赞不已。“首先我提倡不要把运动当成一个任务，要尽可能将它视作生活中常态化的活动。”徐梦桃表示，“每类人群都有自己适合的健身器材，比如漫步机适合增强核心力量，转圆盘可以缓解肩颈疼痛，蹬腿器能增强大腿小腿力量，符合大众日常锻炼需求。”运动是一种积极的力量，更是传递着生活的乐观态度。“看到这么多大爷大妈、小朋友和家长一起坚持运动，真的让我很感动。”桃子姐一边热心帮忙指导人们正确使用健身器材，一边自己也加入了PK比拼的运动行列，不愧是冬奥冠军桃子姐，带你乘风破浪动起来。婚姻与家庭
