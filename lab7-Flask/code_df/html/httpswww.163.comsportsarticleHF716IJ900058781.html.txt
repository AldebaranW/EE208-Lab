曝国米股份将被抵押 张康阳债主已着手寻找新买家用微信扫码二维码分享至好友和朋友圈网易体育8月20日报道：据意大利财经媒体《24小时太阳报》报道，如果主席无法及时偿还，那么橡树基金将获得被抵押的国米俱乐部股份。不过橡树基金并不想接管国米，他们已经委托高盛集团向潜在投资者兜售国米股份。张康阳主席在2021年5月以国米股份为抵押向橡树基金贷款2.71亿，利率为12%，到2021年底贷款和利息总额达到了2.92亿欧元。根据贷款合同，张康阳主席需要在2024年偿还橡树基金的全部贷款和利息。但《24小时太阳报》指出，考虑到目前张康阳家族的财务困难，已经很难想象张康阳能够按时还贷。如果张康阳无法及时偿还贷款，那么被抵押的国米俱乐部股份将被橡树基金获得。过去意甲有过类似的先例，米兰前老板就是因为无法及时偿还贷款，导致米兰俱乐部在2018年被埃利奥特基金掌控。之后几年中埃利奥特基金执掌米兰并让红黑军团成功复兴，上赛季米兰夺得意甲冠军，今年夏天埃利奥特基金将米兰俱乐部卖给了红鸟集团。不过《24小时太阳报》表示，橡树基金没有埃利奥特基金这样的耐心，他们并不准备接管国米俱乐部，一旦获得国米股份橡树基金希望转卖出去尽快获得投资收益。在橡树基金的委托下，高盛集团即将向潜在投资者发送一份预售文件，希望为国米找到新下家。媒体还指出，张康阳也在试图提前寻找买家，希望在贷款到期前以更高的价格出售国米俱乐部，避免遭遇和李勇鸿类似的命运。尽管多次遭到否认，但事实上国米俱乐部已经被挂牌出售一段时间。去年英国BC Partners私募基金曾有意开价7.5亿欧元求购国米，但没能达成协议。如果除掉国米俱乐部的负债，那么张康阳只能得到3.5亿欧元，仅略高于橡树基金的贷款，这是他无法接受的。张康阳希望以至少10亿欧元出售国米俱乐部，但这非常困难。论史记 2023-11-01 09:57:58快闪足球 2023-11-01 22:35:10快闪足球 2023-10-31 22:21:22球文速递 2023-11-01 15:13:24条条爱侃球 2023-11-01 21:25:06陈三谈教育 2023-11-01 14:27:11福酱侃球 2023-10-31 20:27:59小火箭爱体育 2023-11-01 13:51:53肥强侃球 2023-11-01 12:40:51德融视讯 2023-11-01 13:52:10叶檀财经 2023-05-15 21:08:20我爱英超 2023-10-31 21:30:10中国新闻周刊 2023-11-01 13:38:46星耀国际足坛 2023-10-31 12:42:49财联社 2023-11-01 15:43:45青岛文艺 2023-11-01 14:55:51叁炮体育 2023-11-01 11:38:29安宜生活汇 2023-11-01 00:07:55球文快递 2023-11-01 06:45:03华尔街见闻官方 2023-06-19 21:39:20侃球熊弟 2023-11-01 09:37:23罗森内里南看台 2023-11-01 10:42:29快闪足球 2023-10-31 22:45:28Super历史 2023-10-31 10:51:31财联社 2023-11-01 09:55:27伽煜寺简写 2023-11-01 20:23:47每日经济新闻 2023-11-01 22:58:06第一军情 2023-10-31 10:03:43华尔街见闻官方 2023-06-19 20:26:19直播吧 2023-11-01 06:20:26东方豪侠 2023-11-01 08:11:10福酱侃球 2023-10-31 23:43:22有事问彭叔 2023-11-01 14:22:50网易体育 2023-11-01 12:36:21懂球帝 2023-11-01 01:55:23澎湃新闻 2023-11-01 20:27:13流苏晚晴 2023-11-01 20:35:25财联社 2023-11-01 15:20:38网易体育 2023-11-01 21:48:58静海融媒 2023-11-01 14:57:48
