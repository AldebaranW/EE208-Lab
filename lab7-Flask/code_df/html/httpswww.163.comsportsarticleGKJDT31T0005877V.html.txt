三战赢116分，辽宁男篮全运会上吊打对手，郭艾伦都扣篮了，冠军稳了吗？用微信扫码二维码分享至好友和朋友圈40分赢陕西，55分赢湖北，21分赢，全运男篮3连胜锁定小组第一，强势杀入四强。小组赛第三场对北京，第四节还剩3分49秒时，、被换下场，以72-48大比分领先，比赛就此大局已定。3场比赛净胜对手116分，的小组赛完全不是一个级别，看过辽宁队的比赛，给人一种成年人打孩子的感觉。主观感受之外，让我们用客观数据来看一看，辽宁男篮的优势有多大。在小组赛阶段，辽宁队共有4项数据名列第一：1、。在全运会整场40分钟的赛制下，这个攻击力显得异常强大。2、辽宁队小组赛累计净胜对手116分，，这一数据不仅高居榜首，而且是排名第2的队（23.5分）的1.6倍，可见辽宁队的赢球有多么轻松。3、在投篮命中率一项上，辽宁队的命中率高达51.8%，是全运会8支球队中唯一超过50%的，排名第2的队和第3的浙江队，命中率分别只有45.9%和42.6%。4、助攻榜排名第一的同样是辽宁队，他们的总助攻数为59次，场均助攻19.7次，表现出极强的进攻整体性。比赛有多轻松，看下面这个镜头就可以了：对阵湖北队比赛的第三节，郭艾伦抢断汤杰一条龙杀入篮下，以速度见长的汤杰也放弃了追防，整个半场只有郭艾伦一人，打高兴了的郭少用一记单手劈扣给这次进攻画上了圆满句号。要知道在郭艾伦参加的11个赛季的CBA赛场上，郭艾伦的扣篮一共只有21次。这也难怪湖北队主教练代勇，在看到郭艾伦扣篮后，做出了挠头苦笑的动作和表情了。一周前，在全运女篮赛场上，我们曾经看到过这样看过切菜的场面，那是以女篮国家队为班底的奥运联合队吊打缺少国手的各支省队。虽说全运男篮是省队之间的抗衡，但看阵容，如果把辽宁队与小组赛3个对手对比，比喻成国家队打省队也能说得过去。周琦、郭艾伦、韩德君、赵继伟、张镇麟，这5人都是国家队的球员，而且还是一套组合。他们的小组赛对手，阵容如何呢？东道主陕西队，基本是以NBL和大学生球员为班底，名气最大的是曾在北京青年队打球的张玉，以这个阵容来打全运会，重在参与的意义更多一些。湖北队阵中，只有顾全这一名前国手。人气最高的球星是汤杰，并非由于其实力多么出众，而他的独特的球风和球场上经常搞怪的场景，更容易引人关注。至于北京队，全运阵容中没有一名国手，丘天、常林、田宇翔等人都是CBA时期北京男篮的替补，唯一有看点且即战力不错的是曾凡博，可惜他在之前的比赛中伤了，对阵辽宁作壁上观，无缘与张镇麟的“未来国产第一锋线PK”。而辽宁队的先发五虎，非但是国手级球员，而且恰好复制了今年3月CBA北区全明星队的首发阵容。以一套全明星的先发阵容，来参加全运会的比赛，也难怪辽宁男篮有能力实施降维打击。在球员个人数据方面，辽宁队的几名主力也显露峥嵘。郭艾伦场均贡献22分，在得分榜上名列第2，仅次于浙江队的吴前；赵继伟以场均5.7次助攻，在助攻榜上排名第1。张镇麟是辽宁先发五虎中，唯一的U22适龄球员。在CBA赛场，场面劲爆的扣篮就已经是他的保留节目；到了全运会上，这更成为了常规操作。3场小组赛，张镇麟一共送出了8记扣篮，排名扣篮榜的榜首位置。等待张镇麟的扣篮表演，成了观众们看辽宁比赛重要的期待。小组赛的3场球，对于辽宁队来说只是练兵性质。作为全运会男篮的卫冕冠军，接下来的淘汰赛，对于志在连冠的辽宁队才是关键所在。从小组赛阶段的表现来看，辽宁队对于本届赛事准备得比较充分，主力阵容磨合很好，无论是先发球员还是替补轮换球员，都保持了不错的状态。除了五名主力以外，王加冕、刘雁宇等年轻球员，上场就敢于做动作，出了空位机会坚决投，命中率也不错。由于3场小组赛均是大比分吊打对手，因此辽宁队主力球员们，得到了充分的休息：郭艾伦场均23分钟，周琦场均19分钟。这对于比较依赖主力阵容发挥的他们来说，减少主力的损耗是个不错的消息。不过辽宁队的小组赛赢得轻松，很大程度上是因为对手太弱。作为卫冕冠军，辽宁队的目标肯定不是小组赛，争冠才是唯一的目标，要达到这个目标，辽宁队必须跨过另一小组的浙江和。不出意外的话，淘汰赛阶段的两场球，辽宁将先后与浙江、广东交手，这两队的实力都很强，浙江全运队是浙江队原班人马，球员之间的默契程度比辽宁要好，而且在近两年CBA均演练过多场全华班阵容，去年常规赛开局还以全华班拿下了7连胜，广东的阵容就更为强大了。全运会上的广东队，拥有周鹏、赵睿、胡明轩、沈梓捷、刘传兴、徐杰这样的国手级球员。尽管他们不像辽宁队从1-5号位全是国手级的完整阵容，但阵容的深度胜过一筹。小组赛前两场，广东均实现了12人轮换11人得分，且球员上场时间比较平均，他们在小组赛阶段同样保存了实力。过去两届CBA总决赛，广东均战胜了辽宁夺冠，今年的总决赛更是由杜锋VS杨鸣带队直接交手过。从执教经验以及临场指挥水平来看，杜锋占据明显的上风，加上连克对手的心理优势，广东队绝对是辽宁队力争卫冕的头号劲敌。开局阶段辽宁队赢得比较轻松，球队球员在收获自信的同时，也难免会有所松懈。淘汰赛遇上强敌后，比赛的容错率大大降低，这就更加考验辽宁球员的心理和调整能力。小组赛时未暴露的问题，可能会在淘汰赛爆发，教练组的临场应对也变得格外重要。此外，对阵强队时辽宁很可能使用短轮换，对主力球员的发挥要求较高。小组赛末轮对北京，周琦在首节一度因伤退场，张镇麟上半场状态不佳仅得3分，辽宁在第二节一度落后7分。对阵北京尚且如此，碰上浙江和广东更是考验重重。StartFragmentEndFragment不出大的意外，本届全运会的冠亚军决赛，仍将上演辽粤争霸的局面。辽宁队的首发阵容占据上风，广东的板凳深度更胜一筹，双方将贡献一场五五开的激烈对决。兵哥篮球故事
