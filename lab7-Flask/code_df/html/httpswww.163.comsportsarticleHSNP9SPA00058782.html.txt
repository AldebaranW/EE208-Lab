范可新：爬坡难下坡爽，别忘刹车|奥运|短道速滑|奥运冠军|杨扬|王濛_网易体育
2023-02-04
范可新：爬坡难下坡爽，别忘刹车用微信扫码二维码分享至好友和朋友圈更新了个人简介，“世锦赛五冠王”前面添上了长长的一串——2022北京冬奥会混合团体接力冠军。走下冬奥会最高领奖台，她二十年的冰上生涯摁下暂停键，开启了新生活。过去的三百多天里，范可新作为优秀运动员代表，参加了各种体育论坛和商业课程，试图探寻体育的本质，也在重新审视自己与短道速滑之间的关系。没有了日复一日的训练，范可新开始尝试不同的运动。蹬上公路自行车后，这位东北姑娘迅速“入坑”，很快就将自己的骑行距离刷到了100公里，她在车座上一边看风景，一边思考。她发现自行车和人生相同，爬坡的路途都是艰辛，只要登上顶峰，下边的路就会顺畅很多，但重要的是，把握刹车。“夺冠后亲吻冰面，那是吻别的意思吗？“范可新被无数次问及那个名场面，她总说，那只是在传递她对这项运动的感情。而夺冠一年后，这位三十岁的老将坚定了自己的想法，她不想就此结束冰上生涯，即使就做个陪练，她也要走完米兰周期。开始结束12年了，范可新终于又能回家过年。2023年春节前，范可新坐上了离京的飞机，再换乘趟高铁以及小车后，终于抵达老家七台河。黑龙江七台河，被称为煤都，在这片富含矿产资源的土壤中，走出了中国速滑领域最闪耀的金子——和，她们是两代中国速滑选手的旗帜，速滑是这座城市的另一张名片。回家。大平米的楼房，户型板正，窗户面积大，冬天的太阳能照进厅里很远。在省队的时期，范可新坐火车回乡，到站后先得给父亲打个电话，问清楚家搬去哪了。多年来，她和父母辗转于不同的铁皮房子和地下室，有限的的空间还要充当工作间，父母在这里修车配钥匙，一块门板稍加改造，便成她的床铺。范可新从不讳言那段过往，她曾带着摄影团队重返住过的地下室，笑着讲述年少时的生活。她坦然地告诉界外编辑部，小时候坚定滑冰的原动力之一，就是奖金。“第一次参加全国小学生比赛，我把所有的金牌都得了，当时应该10岁，得了1000多块钱，那个时候1000多块钱可是太多了。小的时候很单纯，我想拿到冠军了，我是不是就能得奖金了，那就可以挣钱了。”穷人家的孩子早当家，范可新从早就对钱有概念，是因为她想为这个家贡献自己的力量。范可新曾经的家很快，范可新划出了七台河，进入省队，并且得到了国家队的召唤。父母的生活也得到了相应的改善。时隔十二年再回老家过年，父母再也不用为生计发愁，他们不用四处淘换衣服，各大品牌送来的都是尖儿货，母亲学起了乐器葫芦丝，父亲早晚下楼遛狗。奥运改变生活。过年走亲访友，朋友聚会，大家端起酒杯，都要先跟“奥运冠军”喝一个。北京冬奥会成了饭桌上避不开的话题，范可新笑着讲起了一年前的故事。团体混合接力，中国队顺利晋级半决赛。由于半决赛对手整体实力不强，为了保存范可新的体力，教练组决定派出年轻选手张雨婷出战。短道速滑的赛场瞬息万变，任何一个细微的动作都能影响比赛的结果，充满未知既是这项运动的魅力也是危险所在。半决赛，所有人担心的事情发生了：由于交接棒失误，中国队位列小组第三无缘晋级。“当时休息室很安静，大家的心情特别沉重。我们每天都非常努力地训练，有时一天10个小时，晚上还要加练，不希望这样就结束，”张雨婷回到休息室后开始掉眼泪，其他人坐在一块，也不知道该说什么，“这10分钟好像比我备战的4年更漫长。”就在众人近乎绝望之际，范可新猛地发现，屏幕上出现了美国、俄罗斯两队犯规，中国队递补晋级！一时间，有人跪地庆祝，有人兴奋地拍桌子，大家兴奋地拥抱在一起，仿佛夺冠一般。2022年2月5日，北京冬奥会短道速滑男女2000米混合接力决赛，范可新与曲春雨、任子威、武大靖和张雨婷组成的中国队以2分37秒348获得金牌，为中国代表团摘得北京冬奥会的首枚金牌。“当颁奖仪式现场司仪播报出‘China’那一刻时，内心曾经遭受的困难与辛酸仿佛全都不重要了。”范可新回忆起夺冠的时候，眼神里放着光，只有她自己知道，为了这枚奥运金牌，她付出了多少。上一个奥运周期，身为五届短道速滑世锦赛500米冠军，踌躇满志的范可新在平昌冬奥会颗粒无收，士气受挫的她一度在退役边缘徘徊。“平昌之后，自己负能量特别多，我已经没有自信了，教练说上场比赛我的眼神就出卖了自己。”回忆起那段痛苦的时光，范可新表示当时特别需要别人的开导，那些话她可能也听不进去多少，但这个过程对她意义重大。在多位前辈和队友的开导和帮助下，范可新重拾信心，继续逐梦。短道速滑混合团体接力是北京冬奥会决出首枚金牌的项目，主场作战的中国队背负着拿下首金的压力，全队上下都紧绷着一根弦。不走运的是，范可新在冬奥开幕前十天受了伤。“我太想拿奥运冠军了，当时做单腿蹲练爆发力，总想加点重量，结果第二天就发现腰弯不下去。”在医生的建议下，范可新停训了一天，看着队友们在冰场上备战，她的心里五味杂陈，甚至担心就此与奥运无缘。运动员服用药物受到诸多限制且时间紧迫，范可新只能接受辅助性治疗。有一晚，扎完针后，疼痛难忍的她不禁落泪。范可新说，能坚持这么多年，就是意志力在"顶"着她。这些故事，范可新不仅讲给自己的亲戚朋友，在去年一整年的休整期，她也将自己的故事分享给读者、运动员和体育行业从业者。界外编辑部在懒熊体育的产业论坛上见到了范可新，不同于过去二十年紧身衣和头盔的一贯形象，范可新妆发精致，搭选了手表、眼镜等配饰，就连马甲系哪颗扣子都很讲究。坐在她旁边的是前中国女篮队长邵婷和主持人刘语熙，她们就讲好女性体育故事展开了讨论。范可新化妆了新洞察的一篇文张名为《世人皆羡谷爱凌，我却独爱范可新》，将她视作中国本土教育的典型。认为范可新严肃的、苦涩的，没有光环加持的人生，是大部分平凡人成功路径的缩影。前奥运冠军徐莉佳也在范可新身上找到了自己的影子，她对于职业所倾注的情感同样真挚但复杂。一开始枯燥的练习根本体会不到运动的快乐，但在日复一日的练习中发现它成为了生命中的一部分。整整一年的休整期，范可新尝试了多种运动，这些竞技专项外的运动，让她体会到了简单的快乐。夏天打飞盘玩陆冲，她尝试用不同的弧度将飞盘传给前方的队友。冬天去滑雪，再摔了两趟后，她琢磨出滑冰和滑雪的区别在于重心的运用，落在双腿还是单腿。即使是这些快乐局，她也要求自己，今天要比昨天有进步。“入坑”骑行她入坑骑行，很快就将骑行距离刷到了一百公里。秋日傍晚穿越枫树林收获美景，也会在偶尔想较劲时与摩的狂飙。她享受骑行带来的恣意畅快，更喜欢在骑车时思考的过程。从平缓的路面骑到山坡，阻力越来越大，直至来到顶端。这一切仿佛是她运动生涯的写照：一次次与困难较量，战胜它再迎接新的挑战。她发现自行车和人生相同，爬坡的路途都是艰辛，只要登上顶峰，下边的路就会顺畅很多，但重要的是，把握刹车。“夺冠后亲吻冰面，那是吻别的意思吗？“范可新被无数次问及那个名场面，她总说，那只是在传递她对这项运动的感情。冬奥赛场上，范可新跪下对冰面深情一吻而夺冠一年后，这位三十岁的老将拒绝了快乐的生活，她想重新回归那种枯燥。范可新认为自己有责任做好队伍传帮带。她至今仍记得若干年前在大赛上创造佳绩后收到杨扬鼓励时的喜悦。从杨扬到王濛，再到如今的范可新，队伍的拼搏精神得以在一代代运动间传承。她不想就此结束冰上生涯，即使就做个陪练，范可新也要走完米兰奥运周期。新华社
