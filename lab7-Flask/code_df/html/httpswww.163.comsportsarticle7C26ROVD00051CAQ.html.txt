网易"新生力量"VS李超:要冲击世界冠军真得很难用微信扫码二维码分享至好友和朋友圈网易体育8月22日报道：深圳进入倒计时，在21日进行的的比赛当中，来自首都体育的小将夺得国际象棋男子和混合团体两枚金牌，收获颇丰。在做客网易，参加网易体育与乔丹体育联合举办的大运会特别访谈节目时，李超表示自己的天赋一般，能够拿到两枚金牌与自己后天的努力密不可分，同时运气也是一方面的因素。李超还表示，要冲击世界冠军是非常困难的。   李超：大家好，我是李超。 李超：肯定是相当满意的，这次我觉得也算超水平发挥，运气也不错，很多棋不是特别好赢的最后都让我赢了，这次只和了一盘，对任何起手来讲都是很难做到的，运气非常好。 李超：没有，其实每盘棋都是相当累的，只是对手的心态不一样，大家都是大学生，同龄人，初生牛犊不怕虎，都想赢对方，这时候谁对心理的控制好一点，谁就能走得更远一点，有几盘棋他们都走得有点急，可能使自己的局面变形了，给了我机会赢他们。比较难的那盘是对王皓那一盘，王皓是一号，也是本次比赛得分最高的，不过那时候他心态也有点问题，自己能赢下那盘棋，觉得有可能快冠军了（笑）。 李超：我家里人也都懂棋，在看直播，他们也向我表示祝贺，不过还是要低调一点，因为还有最后一盘，很关键，关系到团体名次，不过还好。 李超：毫无疑问，我觉得这个比赛主要对几个方面比较有考验，首先是你的体力，因为这个比赛一天两盘棋，并且大家都是大学生，同龄人，基本都是睡一觉第二天起来又有精神的那种，国外的人吃的和我们不太一样，体力上更好一点，像我们每天下完棋就相当累了，基本什么事儿也不干，就是睡觉。 李超：有人说下一盘棋相当于打一场拳击赛，我不知道打拳击赛有多累，但下棋真的挺累的，随便下下无所谓，但认真的话，真的挺累的。 李超：没什么秘诀，主要有两点，这次谢军老师在这边，对我心理上的辅导很大，还有我的室友王越，这次比赛的二号，在一号的位置上很久，他在技术上也对我很有帮助，我们俩合作有好几年了，可能也是厚积薄发的结果。 李超：我俩最近很少碰上，都很熟悉了，又进一个房间，都是太原人…… 李超：基本上我俩干点啥彼此都知道。 李超：一直是我的老师，无论我下什么成绩，他都是看着我起来的，从小把我带到现在，无论是在技术还是心理上，尤其是在心理上给了我很大帮助，使我变得慢慢成熟，可能有时候我会骄傲一点，他就会把我翘起的尾巴打下去。 李超：会的，我还挺怕他的。 李超：会的，他以前是北京体院的院长，我也在北京队，他一直管着我嘛，毕竟是世界冠军，也在那个位置，他严肃起来真的挺怕他的。 李超：也没有什么奖励，他能表扬我就挺不错了，因为他挺严厉的，不过这次还算没给他丢脸。 李超：还好吧，没什么太大改变，我也是今年刚入学，因为是体育特长生，所以入学比一般人早一点，但我觉得没有太多改变，后几年可能会体会比较深一点吧。 李超：六到八个小时。 李超：没有，我觉得我智力特别一般，天赋也特别一般，能下成这样真的是靠后天，因为国际象棋里聪明的人太多太多了，像王越，我们住一屋，很多问题我一问他他就直接告诉我答案，我俩的智力根本不是同一水平线上的，很多东西他反应起来比我快很多，那没办法，只能用更多时间训练了，因为在我们圈子里有天赋的人太多了，并且这个圈子的的人真的是非常聪明的圈子，而我在这个圈子里的智力绝对是中下水平。 李超：没有没有。 李超：也没有吧，我能来下这次比赛，无论是学校还是国家队都给了我很大帮助，学校的老师也会在我成绩要求上给予一些照顾，有利有弊，毕竟我还兼着国际象棋这一项目，可能我们这个群体不是特别正常的一群大学生，只是比一般的大学生在国际象棋方面付出的时间多一点，在学习上面落下一点，所以学校领导也给了我们很大照顾。 李超：可能每个人聪明的地方不一样，单纯从智力来说我真的是不聪明，但可能逻辑推理还凑合吧，我也挺喜欢杀人游戏的，玩的时间长了就慢慢会了，要是比智力的话，就是叫几个人一起，比一个陌生项目，这样公平一点，你比别人玩得多，那你肯定熟悉一点，就像下棋一样，不是我下得有多好，就是比别人下得时间多一点。 李超：我比较喜欢当警察，藏着。 李超：希望是这样。 李超：我比较喜欢弄电脑，捣鼓软件，尤其我们这个项目平时训练都需要用电脑，所以自己也希望在计算机方面掌握一些知识，为了更好配合我以后的训练。 李超：还算好吧，我的电脑在国际象棋圈里还算不错。 李超：也挺烦的（笑）。 李超：也挺喜欢的。 李超：希望能有发展吧。 李超：崇拜的人肯定有，印度的阿南德，挪威的卡尔森，这些都是是我无法企及的高度，在我看来。 李超：真的是天赋不够，你说要冲击世界冠军，我觉得真的很难，因为到达一定高度，大家的用功程度就差不多了，有可能你的天赋更高一点，就会让你跨越瓶颈，而我只能尽自己最大努力做到最好吧。 李超：对，希望是这样吧。 李超：好的，谢谢你。 谈资成都 2023-10-31 19:45:30看透成都 2023-11-01 17:02:16深度解读历史 2023-10-28 21:25:20莫叔运动世界
