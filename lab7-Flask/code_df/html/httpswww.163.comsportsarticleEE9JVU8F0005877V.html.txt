连续3年总决赛4-0! CBA应避免"大结局"留住悬念用微信扫码二维码分享至好友和朋友圈网易体育5月3日报道：时间5月3日晚，CBA总决赛第4场103-98战胜，总比分4-0轻松夺冠。这已经是最近3个赛季CBA均出现总决赛4-0横扫的局面，CBA总决赛的悬念性不足，一定程度影响到联赛的进一步推广。今晚的比赛广东客场战胜新疆，从而实现了对于对手的4-0横扫。这一结果不但令广东队夺得了队史上第9个总冠军，而且令他们完成了复仇——两年前即2016-17赛季的CBA总决赛，同样是粤疆两强相遇，结果拥有、亚当斯、这“三巨头”的新疆，以4-0横扫广东首次夺冠；如今，以、马尚、威姆斯三猛将为核心的广东，以彼之道还施彼身。上赛季的CBA总决赛由辽宁对阵广厦，最终实力技高一筹的辽宁连赢四阵，以总比分4-0击溃广厦，同样是夺得队史上首个总冠军。如此一来，算上今年的结果，CBA总决赛已经是连续三年呈现4-0吊打夺冠这样一边倒的结局。原本被预期两强遭遇、势均力敌的总决赛，却显得如此强弱分明、令人感到乏味。纵观最近三年的总冠军球队，共同点是组建了超级豪华阵容：前年的新疆以布拉彻和亚当斯双外组合火力全开，内线有防守大闸，加上李根这个爆分强点无人可以对位；去年的辽宁在保留郭艾伦和哈德森“后场双枪”基础上，签下巴斯补上大外援短板，再有赵继伟、李晓旭等板凳强援；今年广东更是以易建联为核心，先后配备威姆斯、马尚、比斯利这三个超级外援，周鹏、赵睿等角色球员同样给力。阵容的失衡性令联赛出现了一家独大的局面。作为一个以商业化运作为主的联赛，如何通过合理的政策调整，最大程度保持比赛的悬念性是相当重要的。职业化已相当成熟的NBA以“工资帽”和“奢侈税”方式，长期遏制了超级军团的诞生，在平衡性方面取得成功。但随着近年勇士队巨星扎堆的特殊团队，NBA早早“大结局”也导致了收视率明显下滑。CBA对于这种现象也应当引以为戒，采取有效的预防措施。King迪哥侃球 2023-10-29 17:52:37田先生篮球 2023-10-31 11:41:06历史名匠锁 2023-11-01 10:37:57情感召唤师 2023-11-01 09:54:56历史名匠锁 2023-11-01 10:26:29篮坛酷侃 2023-11-01 12:47:40阿木侃球 2023-10-30 18:39:48君马体育 2023-10-31 20:11:19听闻说史 2023-10-30 17:25:48多特体育说 2023-11-01 21:56:23多特体育说 2023-11-01 22:21:25篮球大陆 2023-11-01 22:44:58多特体育说 2023-11-01 22:45:07德融视讯 2023-11-01 13:52:10独特的文史 2023-10-29 20:51:20看度新闻 2023-11-01 15:38:52绯雨儿 2023-11-01 13:10:03毒液体育 2023-10-31 21:40:49青岛文艺 2023-11-01 14:55:51东球弟 2023-11-01 21:43:09理工男评篮球 2023-11-01 22:01:44篮球资讯达人 2023-11-01 23:12:04东球弟 2023-11-01 08:52:34侃球熊弟 2023-11-01 09:37:23新娱乐圈看点 2023-11-01 10:17:54网易体育 2023-11-01 21:37:10网易体育 2023-11-01 21:31:49后厂村体工队 2023-11-01 09:35:41东方豪侠 2023-11-01 08:11:10东球弟 2023-10-30 12:50:05网易体育 2023-11-01 12:36:21澎湃新闻 2023-11-01 20:27:13财联社 2023-11-01 15:20:38财联社 2023-11-01 09:55:27网易体育 2023-11-01 21:48:58央视新闻客户端 2023-11-01 15:59:20滑县市场监管 2023-11-01 20:50:41静海融媒 2023-11-01 14:57:48星晨视频 2023-11-01 15:02:49澎湃新闻 2023-11-01 16:06:54
