孙铭徽33+6+6广厦再胜辽宁夺赛点 胡金秋12+18用微信扫码二维码分享至好友和朋友圈网易体育4月28日报道:时间4月28日晚，CBA季后赛半决赛继续进行，主场迎战，郭艾伦因伤缺阵；经过四节的较量，最终以87-77击败，总比分2-1领先，拿到系列赛的赛点。四节比分分别为16-18，25-18，20-23，26-18（广厦在前）技术统计：广厦: 33分6篮板6助攻（命中6记三分）、12分18篮板、朱俊龙10分6篮板、威尔哲10分3篮板辽宁:弗格22分5篮板6抢断、赵继伟11分10助攻、付豪11分4篮板、张镇麟10分8篮板赛事回放：比赛开始后，广厦先声夺人，孙铭徽与威尔哲联手命中三记三分，广厦打了一波11-3的开局，逼辽宁叫了暂停。暂停后，辽宁强势反击，李晓旭和赵继伟分别命中三分，随后丛明晨也命中三分，辽宁就这样接管了比赛，打出了一波13-0的高潮，随着首节4分09秒莫兰德跳投命中，辽宁以16-11领先。再看广厦，进攻完全陷入挣扎，还出现失误，直到首节还剩2分29秒时，广厦才终于再次得分，孙铭徽两罚全中，而他们上一次得分还是首节8分42秒，也就是说他们在6分13秒里一分未得。随后孙铭徽又砍下3分，帮助广厦在首节结束时将比分追至16-18。进入次节，卡明斯迅速抛投得分，将比分追至18平，比赛也开始陷入胶着战，双方比分交替领先。不过辽宁的进攻跟首节相比明显下滑，次节前半节仅得5分，广厦逐渐掌握场上的主动权。次节进行至5分56秒，胡金秋在内线强打勾手命中，随后又几乎复制了同样的进攻，跟着朱俊龙跳投命中，广厦以32-23领先9分。随后迅速砍下5分，率领辽宁反击，打出一波11-5的高潮，弗格的反击上篮帮助辽宁将比分追至34-37。随着威尔斯和胡金秋连续得分，广厦在上半场结束时还是以41-36领先。下半场上来，赵继伟命中三分，弗格上篮得手，辽宁迅速将比分追至41平，然后双方打到43平，赵继伟再命中三分，辽宁反超2分，但广厦又将比分打到46平。孙铭徽连续命中两记三分，张镇麟则还了一记三分，辽宁将比分追至51-52。第三节进行至3分44秒，胡金秋跳投命中，跟着孙铭徽再度三分命中，广厦再度领先5分。随后在篮下抢篮板时，莫兰德与许钟豪手臂纠缠在一起，然后抱摔倒地，险些酿起冲突。虽然广厦很快又将分差拉开，但张镇麟在第三节结束前命中压哨三分，帮助辽宁将比分追至59-61。末节比赛，威尔斯上来连投带罚砍下4分，帮助广厦再度领先6分。随后双方上演了飙分大戏，各命中了两记三分。末节进行至4分28秒，弗格跳投命中，将比分追至71-75，但很快胡金秋就用勾手命中回应，跟着孙铭徽上篮得分，朱俊龙两罚一中，广厦领先9分，比分为80-71。弗格连续命中两记三分，赵嘉仁回应一记三分，跟着孙铭徽也跳投命中，将分差拉大到8分，稳住了局势，辽宁也没能再发起反击，比赛也就此失去悬念，最终广厦击败辽宁。双方首发：辽宁：赵继伟、张镇麟、李晓旭、弗格、韩德君广厦：赵嘉仁、孙铭徽、胡金秋、朱俊龙、威尔哲篮球资讯达人
