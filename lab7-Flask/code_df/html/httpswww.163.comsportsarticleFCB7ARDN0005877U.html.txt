锡安前经纪人:锡安父母曾从大学和赞助商收过黑钱|阿迪|吉娜_网易体育
2020-05-11
锡安前经纪人:锡安父母曾从大学和赞助商收过黑钱用微信扫码二维码分享至好友和朋友圈网易体育5月11日报道:是2019级的状元秀，他在NBA赛场有着出色的表现。然而在停摆期间他却陷入了法律纠纷：他的前经纪人-福特在近日爆料，锡安在选择杜克时就收取了不正当利益，之后他还从耐克、两大赞助商中分别获利。巅峰体育（Prime Sports）是位于迈阿密的一家经纪公司，他们原本在去年4月就跟锡安签订了一份为期5年的合同，但在后者宣布参选NBA之后便宣布解约，并将公司告上法庭，要求终止相关合同。锡安和他的团队表示，他与巅峰体育的合同是非法的，因为公司没有获得NBA球员工会的证明，也没有在北卡罗莱纳或佛罗里达州注册运动员经纪人的资格。但公司和福特以违约为名提出反诉，强调他们之前已经签订了一份5年的合同，是锡安违约在先，并要求锡安至少支付1亿美元的赔偿金。根据诉讼律师丹尼尔-沃勒克在社交媒体上的爆料，吉娜在法庭上的一份文件被曝光，里面提到她作为锡安经纪人期间的一些情况：包括锡安的母亲和继父接受杜克大学和耐克的金钱和礼物，直接或间接影响了锡安去杜克打球；母亲和继父还接受了阿迪的经济支持，让锡安在进入大学前一直穿阿迪的鞋打球；在2014年1月1日到2019年4月14日之前，锡安的父母一直从各大经纪人之间接受好处。NCAA一直禁止球员与学校、赞助商有经济往来，这有损大学的纯洁性。因此，锡安收受利益得到了外界的关注。当地的网友表示，锡安这种行为虽然不合规矩，但其实已经成为NCAA的潜规则，因此大概率会不了了之。目前锡安团队方面还没有做出具体回应。经典历史解释 2023-11-01 10:13:01XCin球鞋顾问 2023-10-30 21:29:20体娱小小密探 2023-11-01 11:53:02双一流高校 2023-10-31 19:10:42言安堂 2023-10-29 08:35:33直播吧 2023-10-31 11:18:22历史遗失的真相 2023-10-31 16:03:11潮流有货 2023-10-31 23:11:49英国那些事儿 2023-08-13 23:16:10德融视讯 2023-11-01 13:52:10看度新闻 2023-11-01 15:38:52乘风破浪的海叔 2023-10-31 19:08:01皮皮文章J 2023-10-31 22:24:28青岛文艺 2023-11-01 14:55:51西西职场说 2023-11-01 13:22:06陈三谈教育 2023-11-01 14:32:00篮球显微镜 2023-11-01 08:36:06网易体育 2023-11-01 12:36:21一条柴 2023-10-28 15:35:22体育快播 2023-10-31 19:55:30侃球熊弟 2023-11-01 09:37:23小豆动画屋 2023-10-30 00:39:49天羽看球 2023-11-01 02:50:53网易体育 2023-11-01 21:31:49东方豪侠 2023-11-01 08:11:10东球弟 2023-11-01 21:43:09澎湃新闻 2023-11-01 20:27:13财联社 2023-11-01 15:20:38财联社 2023-11-01 09:55:27网易体育 2023-11-01 21:48:58央视新闻客户端 2023-11-01 15:59:20滑县市场监管 2023-11-01 20:50:41静海融媒 2023-11-01 14:57:48星晨视频 2023-11-01 15:02:49澎湃新闻 2023-11-01 16:06:54中国海警 2023-11-01 13:17:44财联社 2023-10-31 22:10:49财联社 2023-11-01 15:43:45央视新闻客户端 2023-11-01 13:27:42懂球帝 2023-11-01 05:37:33
