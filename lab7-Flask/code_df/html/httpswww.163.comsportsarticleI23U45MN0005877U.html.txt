状元连续4次获月最佳新秀比肩奥尼尔 雷霆新秀得奖用微信扫码二维码分享至好友和朋友圈网易体育4月12日报道：NBA官方宣布，凭借在3、4月份的出色表现，奥兰多魔术队的罗和俄克拉荷马城的杰伦-威廉姆斯分别当选东西部3、4月份最佳新秀。这是班凯罗连续第4个月获得东部月最佳新秀。班凯罗加入到的行列中，成为队史上仅有的两位连续4个月获得月最佳新秀的球员。奥尼尔在1992年11月到1993年2月期间做到这一点。班凯罗在3、4月间出场17次，场均得到20.3分、7.8个篮板和4.2次助攻，有38.7%的三分球命中率。在得分上班凯罗领跑所有NBA新秀，出场时间排名第二，总篮板数和助攻数排名第4。班凯罗这期间有10次得分20+表现，两次得分30+表现（两项数据均是新秀球员最佳），还送出4次两双表现。作为2022年NBA选秀的状元郎，班凯罗在本赛季首发出战72场，场均得到球队最高的20分，还有6.9个篮板和3.7次助攻。在场均出场时间上排名联盟并列第37位，篮板并列第38位。在得分、出场时间上他都领先所有新秀，篮板排在第5，助攻排在第3，罚球命中率第7、抢断第4，盖帽第7。本赛季班凯罗40次得分20+，6次得分30+，两项数据都是新秀中最高的。班凯罗成为NBA过去50个赛季里（自1973-74赛季后）第6位场均送出至少20分、6篮板和3次助攻（至少50场）的新秀。杰伦-威廉姆斯在3、4月首发出战18场，场均得到18.7分、5.5个篮板、4.3次助攻和1.67次抢断，其54.7%的投篮命中率领跑球队。得分、抢断均领跑西部新秀，篮板排在新秀的并列第4位。在3月5日，雷霆队129-119战胜爵士队的比赛里，威廉姆斯得到生涯新高32分，在18场比赛里他有9次得分20+。这是威廉姆斯第2次获得西部月最佳新秀称号，他是历史上第3位曾多次赢得月最佳新秀奖项的雷霆队球员。其他被提名月最佳新秀奖项的球员包括了底特律活塞队的埃维、印第安纳步行者队的马瑟林和内姆哈德、达拉斯独行侠队的哈迪、萨克拉门托国王队的穆雷、波特兰开拓者队的夏普和休斯敦火箭队的-史密斯。熊哥爱篮球 2023-11-01 13:39:11故事奇谈录 2023-11-01 19:56:30体坛正中心 2023-11-01 12:15:20情感小王 2023-11-01 09:02:44严堂侃球 2023-10-31 21:56:37罗说NBA 2023-11-01 09:06:35体坛故事集 2023-11-01 11:27:48直播吧 2023-10-31 22:54:12暴笑鸭鸭 2023-11-01 20:37:57鱼崖大话篮球 2023-11-01 13:08:08体坛故事集 2023-11-01 01:55:02直播吧 2023-11-01 05:48:30康泳哥看体育 2023-11-01 15:06:58聊史 2023-10-28 22:32:24天羽看球 2023-11-01 02:54:00康泳哥看体育 2023-10-30 12:19:33篮球显微镜 2023-11-01 08:36:06原是程老师 2023-11-01 19:13:52德融视讯 2023-11-01 13:52:10小绿说故事 2023-11-01 22:33:39篮球显微镜 2023-11-01 12:02:26看度新闻 2023-11-01 15:38:52熊哥爱篮球 2023-11-01 13:07:15青岛文艺 2023-11-01 14:55:51直播吧 2023-11-01 14:44:11直播吧 2023-11-01 11:41:24西西职场说 2023-11-01 13:22:06陈三谈教育 2023-11-01 14:32:00网易体育 2023-11-01 12:36:21体育快播 2023-10-31 19:55:30侃球熊弟 2023-11-01 09:37:23直播吧 2023-11-01 07:24:26天羽看球 2023-11-01 02:50:53东方豪侠 2023-11-01 08:11:10财联社 2023-11-01 09:55:27直播吧 2023-11-01 07:24:26澎湃新闻 2023-11-01 20:27:13财联社 2023-11-01 15:20:38央视新闻客户端 2023-11-01 15:59:20静海融媒 2023-11-01 14:57:48
