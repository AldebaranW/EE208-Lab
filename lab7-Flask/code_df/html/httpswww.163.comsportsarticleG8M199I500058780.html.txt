我也不是油腻的中年男人用微信扫码二维码分享至好友和朋友圈梦剧场建在寸土寸金的前门大栅栏，与天安门广场的步行距离只有800米。应邀在开业前来参观，他走遍了三层的建筑，穿越和老特拉福德球场同款的球员通道、辨识更衣室里那些不同年代的红色战袍，看到鲁尼倒钩射门的那张经典照片时，他立刻说出了那场比赛的对手和比分。这座4000多平米的建筑展示了曼联的方方面面，球队历史上唯一的中国球员沉浸在红魔的历史中，脸上挂着微笑，就像回家一样，尽管这里还没有任何关于他的元素。开始结束只有面对1:1复刻的欧冠冠军奖杯“圣伯莱德杯”时，穿着阿玛尼大Logo针织衫的董方卓拒绝了合影的建议，他觉得今天的装扮不够正式。在一项计分类VR足球互动游戏开始前，工作人员询问他和经纪人谁先，这位退役5年的前锋主动选择了后踢：“不超越，哪有快乐。”而在曼联俱乐部的几年，这位天才少年被诟病最多的，是不求上进，没有竞争意识。这是他至今都不能接受的。辗转多国联赛最终决定退役时，董方卓想过很多转型方向，甚至有当演员这种和前三十年毫无关联的职业幻想，他想像电影里演得那样，体验别人的生活。@@SectionTitle="天才" Type="ordered" SerialNumber="01"@@刚过完18岁生日的那天晚上，董方卓的妈妈接到大连实德俱乐部的电话，那头第一句话就是：“恭喜大姐啊，曼联定了签小董。”成年人都有些预感。第一次15天曼联队内试训后，曼联方面就与实德高层在曼市的一家餐厅聊起董方卓转会的事宜，董方卓又被延长了15天训练，他依旧在用各种方式进球。但当这一时刻真的来临，董方卓还是觉得不真实，像在做梦。这位以为在实德出场为最高人生目标的无名之辈，摇身一变成为了“天才少年”。所有人都想知道，拿到彩票的，为什么是他？与董方卓签署合同条款时任实德主教练朱波将根本原因归结为他超强的身体素质。在接受中新社采访时，朱波称爱徒为同年龄段的中国球员里最突出的一位。那个年代，中国足球刚兴起留洋风潮：杨晨成为德甲球队法兰克福保级路上的功臣；范志毅戴上了水晶宫队长的袖标；孙继海则稳居曼城一线队主力。虽然不是豪门球队，但在当时的球迷看来，中国球员的留洋路是在稳步上升的，董方卓的出现是一个情理之中的意外——总有一天，中国球员是要像姚明征服NBA一样，征服顶级联赛、征服大俱乐部。董方卓一直坚信，是自己的实力打动了曼联。他从未想过登陆时机是否合适，多年以后，再谈到留洋机会时，他仍然觉得那是一个再正常不过的选择，“那可是曼联”。而从被曼联选中的那一刻起，董方卓天然就要承担起这份期待，他是世界上最成功的足球俱乐部为中国足坛钦定的紫微星，是中国人距离征服世界足坛最近的一个人。实德主教练朱波的话被反复提及，董方卓在曼联试训期“每场都进球”的神迹被无限放大，甚至连他“常常收到女孩子的信”也能写成了报道。同一时间，英国记者也在互相打听：“你听说过董吗？”那段日子，坊间流传着两种声音，一种是董方卓的加盟更像是一桩商业转会，是曼联为了打开中国市场的一种商业行为；另一种是，在主教练负责制的英国足坛，特别是弗格森，他的教练生涯里从来没有因为需要妥协俱乐部商业上的规划，而去签任何球员，是试训里董方卓展现的天赋和潜力让弗格森毫不犹豫地签下了这个东方少年。曼联的官方发布会没有邀请中国媒体，原因是俱乐部认为他们在抢新闻时，提前泄漏了董方卓加盟的消息。本就夹在两种情绪中的董方卓，被架上了一个尴尬的位置。直到2004年1月17日，新华社刊登了名为《弗格森对董方卓寄予厚望 “两三年内”立足曼联》的文章，老爵爷第一次对中国媒体表达了对董方卓的期许，他重强调了董方卓需要一年时间适应异国生活，希望他能主动融入新环境。董方卓随即飞去了欧洲，由于没有英格兰劳工证，他不得不委身曼联在比利时的卫星俱乐部皇家安特卫普。时任皇家安特卫普俱乐部主席保罗回忆，弗格森亲自打电话沟通董方卓的发展规划，他明确表示曼联在合适的时机将会召回董方卓。@@SectionTitle="蚂蚱" Type="ordered" SerialNumber="02"@@在比利时，他开始了独居的生活。洗衣机是坏的、吃冷披萨、错过班车后步行两小时回家，和国内训练基地里的一站式服务相比，董方卓经历着完全不同的场外生活。只有球场是熟悉的。董方卓很快征服了那些质疑他的队友，最开始，他在队内训练中拿球总被踢和排挤，直到他在场上跟对位的比利时人干了一次，他发现队友们开始认可他和接纳他了。董方卓自评是一位欧式的前锋，背身拿球不是他的特点，他喜欢向前突破。在比利时，他将自己的优势发挥得淋漓尽致，他完成了前锋的任务。05/06赛季，他以18球获比乙金靴，成为第一个在欧洲联赛拿到最佳射手的中国人。在接下来的06/07赛季，董方卓延续了此前的火热状态，联赛还未过半，就打进了10个球。凭借在比利时的优异发挥，在半程赛季未结束时，弗格森提前把董方卓召回曼联并放进一线队名单。董方卓在安特卫普国际职业球员联合会在这一年给出了最佳年轻球员的12人候选名单，与董方卓肩并肩的球员有C罗、鲁尼、法布雷加斯、基耶利尼、马斯切拉诺、施魏因施泰格、罗本……曼联将他召回时，英国足坛已经没有了质疑的声音，天空体育的解说员更是亲切地叫他“King Dong”，对他回归赛场的表现充满期待，而国内球迷和媒体则盼望着他在比利时足坛的沉淀能助他在曼联一飞冲天。比利时联赛的经历固然高光，但国内没有转播、没有如今便捷的互联网传播，也没有驻地记者对董方卓进行专门报道，等到他重回曼联，足球记者贺小龙就写到：“董方卓，成为了活在传说里的人。”董方卓成为继孙祥后第二个在欧冠出场的中国人另一位驻曼市的《足球报》记者王康宁回忆，即便在董方卓刚刚回归曼联时，国内记者也鲜少能报道关于他的内容，只有他公开亮相时才有新闻可写。在球场上表现平平的董方卓在场下也将自己封闭起来。当时曼联的前锋线人才济济：鲁尼、C罗、特维斯、罗西……在回忆这些同位置的竞争者时，董方卓特别提到了萨哈，萨哈有多厉害？英超的球员都认定他会接亨利的班。英格兰国脚阿兰-史密斯，因为锋线拥挤，按照弗格森的要求，被一步步改造成后腰去竞争出场时间。“在曼联，我就像个小蚂蚱一样。”时隔多年，董方卓面对网易体育的镜头这样自评。回到曼联的他，在场上要为每一分钟出场时间和一众已经成名的世界级球星竞争，场下曼联严苛的队规让他变得谨小慎微。刚回曼联时，队里曾问他是否需要给他专门配备一个中国厨师，董方卓婉拒了，他觉得吃牛排沙拉也行，他怕外界说他搞特殊。在曼联，同属前锋的董方卓甚至还要与C罗竞争他也不敢提加练的请求，因为在欧洲人的思维里，这有可能会被认为对教练训练计划的不满。虽然他在比利时自己也时常加练，但在曼联，这个当时世界最大的俱乐部里，球员所做的一切细节都会被无限放大，对于一个来自中国的球员，试错的成本过于昂贵。加盟曼联已经过去了一年多，当初对他怀有期待的人开始慢慢失去耐心，眼看着同年加盟的C罗已经成了炙手可热的球星。在曼联的几场季前热身赛，董方卓虽也几次上演过高光时刻，却因为残酷的竞争难觅联赛出场机会。期间，德甲球队曾抛来橄榄枝，曼联老队长基恩也曾想将他带到保级球队桑德兰，董方卓自己也非常想被曼联租借出去寻求出场机会，但在这些邀约面前，合同的条款约束让他无法掌握自己的命运。@@SectionTitle="失控" Type="ordered" SerialNumber="03"@@在比利时的出色表现，让回到国家队的董方卓成为国人关注的焦点。但戳破群体期待的，也正是在国字号比赛中，董方卓在2005年世青赛怒推克劳琛的一幕。国青主帅克劳琛整个大赛期间把董方卓摁在板凳上，到了与德国队淘汰赛时，克劳琛在比赛还有3分钟结束时换上了董方卓。那会比分2：3落后的国青队已经大势已去，濒临淘汰边缘。这样的排兵布阵和最后遗憾出局的结果，彻底点燃了年轻气盛的董方卓，为了世青赛，他忍着脚部骨裂打封闭参赛。当克劳琛前来安慰时，坐在替补席上郁闷的董方卓头也没抬一把推开了他。这是董方卓耍大牌的负面形象第一次见诸报端。虽然赛后董方卓向克劳琛诚恳地道歉，但当时国内媒体报道的内容是：“拍摄到这一幕的美国摄影记者吓了一跳”，“曼联回来的少爷瞧不起国家队教练”，至此“耍大牌”成为了董方卓身上继“天才少年”后的第二个标签。在联赛中第一次为曼联出场，那场与切尔西的比赛，将董方卓拉下了神坛。赛前在更衣室得知自己将为曼联首发上场的董方卓没有时间消化这个决定，他只能硬着头皮走上球场。但正如大家看到的那样，他搞砸了首秀，或者说董方卓并没有发挥出自己真实的水平。回国后董方卓曾无数次面对媒体复盘过这场比赛，事到如今他已经可以波澜不惊地承认当年自己是“懵着踢完全程”。而当年，知道自己搞砸了的董方卓并没有如此坦诚地面对媒体。2008年北京奥运会男足比赛，同组对面那支比利时队的青年军，大多在比利时见识过董方卓的厉害，董方卓占据着心里优势，却因为伤病没有得到足够的出场时间。对阵新西兰，有伤在身的董方卓在78分钟替补登场，随后他接陈涛在边路的精妙传中，用一粒精彩的头球破门，帮助中国避免了失利的尴尬，打入了中国男足在世界大赛第一粒也是至今为止唯一一粒入球。除了这个高光时刻，董方卓在国字号球队没有太多的亮点，与球迷们幻想的救世主有不小的差距。那些董方卓得罪过的媒体也没让他在场下好过，曾经董方卓有“天才少年”光环时，记者们不敢写他的负面内容，生怕他变成下一个姚明后，被拉进黑名单。但今时不同往日，董方卓在曼联的生涯几乎看得到尽头，那些从董方卓身上讨不到新闻的人几乎人人都要踩上他一脚。迟到、耍大牌、不自律、懒、生活奢侈，编造的假新闻满天飞，他很想找个场合跟所有人解释清楚，但却发现越描越黑：“非得看到曼市凌晨4点的太阳才算上进吗？”在那个没有微博和公众号、主流媒体掌握话语权的时代，董方卓连为自己发声的渠道都没有。天才少年变成了中国足坛名声最差的人，直到今天在网络搜索董方卓，仍有“脖子以下世界级”的恶语。@@SectionTitle="受伤" Type="ordered" SerialNumber="04"@@有几个人知道亚美尼亚国庆日是哪一天？董方卓是其中一个。9月21日，董方卓被对手恶意出击的门将打掉了门牙，眼眶和膝盖也都受了不同程度的伤。晚上回到宾馆，吞下止疼药的同时，窗外响起了爆竹声。他看着漫天的璀璨，一年半没有回国的董方卓终于做出了决定：结束了，就到这里吧。离开曼联后，他回到大连，球队还叫实德，但他已不是曾经的天才少年。一整个赛季，他颗粒无收。他还是不甘心，想着要证明自己，应邀加盟波兰国内豪门莱吉亚，在度过和俱乐部短暂的蜜月期后，因为出场时间的不稳定，他转战葡超球队波蒂莫嫩塞。但不久后，俱乐部就遭遇经济危机，连换三任主教练，工资顶薪的董方卓虽然刚在杯赛打进一球，却因为薪资的问题不得不离开球队。为了通过踢欧联杯重回欧洲主流联赛，董方卓又加盟了亚美尼亚球队，并在当年杯赛决赛中打进一球，帮助球队夺冠，获得欧联杯资格。但当希望就在眼前时，他的膝盖又伤了；回到国内后，面对脚伤，为了尽快比赛，他接受了俱乐部的建议，选择进行针刀手术，却因此葬送了自己的职业生涯，被迫退役……董方卓手术刚回国那段时间，是董方卓生涯中最惨的一段日子，他认命地接受了自己无法重回顶级联赛的事实，前几年与国内舆论糟糕的相处，也让他难对国内的公众敞开心扉，在球场上，他更加失意，拿着队内最高的工资却连场都上不了。媒体反反复复提起曼联让他烦躁，仿佛这个世界只有前曼联球员董方卓，他们一遍又一遍总结董方卓留洋失败的经验，然后重操起那些真真假假的爆料，以此来证明董方卓因为不会做人、态度不好而成为曼联的弃子。然而一个残酷的真相是：中国的天才少年到了世界上天才少年最多的地方，人们期望着他能将天赋升级成常态，再在这种极富天赋的常态中找到另外的天赋，这对他来说太难了。2017年，董方卓接受了懂球帝的专访，那是他第一次决定对公众敞开心扉，他劝自己：“都退役了，没什么不能说的。”采访中他仔细回答了每一个曾经避而不谈的问题，把自己整个职业生涯的坎坷公之于众。报道出来之后，不少球迷给他的微博发了私信来道歉：“曾经骂过你，对不起”，看着这些道歉，董方卓竟然有一丝想哭：“这么多年了，嘴上一直说着不在意，但突然发现这么多人来跟你道歉，还是觉得很感动。”在社交网络，董方卓甚至主动拿与对阵切尔西的那场比赛调侃自己他很理解国内球迷的失望，但他达不成那样沉重的期待。董方卓坦诚地告诉网易体育，在曼联，他和那些顶级前锋确实存在着差距，他甚至将那些突发的伤病也算作实力的一部分，“如果我训练中多进几个球，那也不会去踢预备队的比赛。”回看这些经历，董方卓对伤病充满无奈，好像每一次看到希望时，命运总会告诉他：对不起，只是玩笑罢了。被问到是否接受伤仲永的说法时，董方卓突然抬高音量：“说的没错啊，我就是伤了，是‘伤’仲永啊”。@@SectionTitle="重来"
