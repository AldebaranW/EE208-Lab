2018常州西太湖半程马拉松赛新闻发布会今日召开|跑者_网易体育
2018-09-27
2018常州西太湖半程马拉松赛新闻发布会今日召开用微信扫码二维码分享至好友和朋友圈9月27日上午10:00，2018西新闻发布会在常州富力喜来登酒店正式举行。江苏省体育竞赛管理中心主任陈小平，常州市人民政府副秘书长黄建德，常州市体育局副局长、调研员李晓华，江南农村商业银行副行长张建军，江苏常奥体育发展有限公司董事长陶婷婷以及赞助商代表、代表、志愿者代表、医疗代表和各界媒体朋友出席了本次新闻发布会。会上发布了本届常马的各项筹备工作，以及跑者最关心的参赛装备、完赛奖牌、保障服务等信息。2018年常州西太湖半程马拉松将由江苏卫视全程直播，共有来自全世界20000名跑者齐聚龙城常州。作为常马的五周年，组委会也为所有跑者准备了各种五周年定制福利，五周年徽章、五周年完赛毛巾以及与天宁寺联名推出祈福完赛签，给大家带来更佳的参赛体验，打造5与伦比的五周年。最美奖牌 !vml][endif]2018常州西太湖半程马拉松设半程和迷你跑终点相同，10公里终点为花博园东北门。今年半程项目沿用17年比赛路线，十公里以及迷你跑项目沿用16年比赛路线，避免不同项目选手的对冲问题，实现了不同项目的参赛者分流，为参赛选手提供了更佳的参赛体验。同时，就报名情况看，本次常马精英选手占所有选手的10.4%，同时组委会还特别邀请了12位专业跑者，希望将常马打造成为全民性与专业性相结合的路跑赛事。相较于前几届，今年常马在补给设置上比往年更丰富，除了常规的香蕉、面包、香肠等，在装备包中增添了方便面，给所有跑者更好的能量保障。同时，组委会考虑到外地跑者的住宿及到达问题，本届常马特别推出“常马商旅套餐”，并安排接驳车辆往返。合作酒店在比赛当天，也根据跑者的需求，将早餐就餐时间提前至6:00，并免费为参赛选手延迟退房至下午14:00。对马拉松赛事而言，安全保障尤为重要。今年，组委会将进一步完善和强化救援保障机制。本次赛事三个项目赛道上共设有12个医疗站，在半程项目上每100米安排一名医疗志愿者服务，同时为赛事配备30台AED设备供医疗站及移动救护医生使用，并有服务过百场专业马拉松的医疗保障团队全程待命。同时，赛前组委会将通过多种形式提醒每一位选手如何科学备战，量力而行，将参赛风险降至最低，享受比赛，拥抱健康。对所有的志愿者也将提前集中培训，保证给跑者最好的服务。感谢所有陪伴常马一起成长的跑者、医疗、安保、救援、搭建等的各方支持，常马的五周年，等着你我一起来见证！10月14日，2018常州西太湖半程马拉松赛，我们不见不散！谈古论
