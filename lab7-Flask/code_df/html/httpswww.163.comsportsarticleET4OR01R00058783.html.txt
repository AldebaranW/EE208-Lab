太湖佳绝处最美在滨湖 2019蠡湖半程马拉松圆满落幕|滨湖区|无锡_网易体育
2019-11-04
太湖佳绝处最美在滨湖 2019蠡湖半程马拉松圆满落幕用微信扫码二维码分享至好友和朋友圈11月3日8:00，2019环国际在市大道隐秀路路口鸣枪起跑。本届赛事由中国田径协会、无锡市区人民政府主办，无锡市滨湖区文化体育和旅游局、汇跑赛事承办，得到了江苏省田径协会支持。出席发枪仪式的领导和嘉宾有：江苏省体育局竞赛管理中心主任 吴超 先生无锡市委主题教育第二巡回指导组副组长 朱俊 先生无锡市体育局局长 黄浩然 先生无锡市文化广电和旅游局副局长 李晓红 先生滨湖区委书记 许峰 先生滨湖区人民政府区长  陈锡伦 先生滨湖区人大常委会主任  许新宇 先生滨湖区政协主席 刘洪兴 先生滨湖区委副书记、组织部部长 殷毅 先生滨湖区委常委、宣传部部长 倪守红 先生滨湖区委常委、政法委书记 张爱军 先生滨湖区人民政府副区长、市公安局滨湖分局局长 钱锡忠 先生滨湖区人民政府副区长 徐新宇 女士无锡汇跑体育有限公司总经理  李长征 先生美的集团洗衣机事业部品牌市场模块负责人  徐健 先生经过激烈的争夺，来自肯尼亚的PETER MURIMI MUTHIKE夺得半程马拉松男子组冠军，成绩为1小时08分41秒；女子冠军则归属于肯尼亚的Caroline Jeruto CHESIR，她的夺冠成绩是1小时15分02秒。在中国马拉松遍地开花、琳琅满目的形势之下，环蠡湖国际半程马拉松通过精心的组织和优质的服务，仅仅用了三年的时间就斩获了中国田径协会“金牌赛事”称号，成为中国马拉松顶级赛事。今年赛事报名开启之后，总共有20089名跑者完成报名，赛事也首次采用抽签方式产生了各个赛项的总计13000名参赛选手。与此同时，赛事启用了大受跑者好评的“退出与候补机制”，用更加人性化的赛事服务，获得了广大跑者的一片赞誉。对于每一位秋日来到无锡参赛的跑友来说，蠡湖半马优美的赛道风光无疑为赛事增色良多。今年依旧延续了景点线路设计，跑友在21.0975公里的赛道上一路穿越蠡湖之光、十里芳径、湿地公园和太湖之星，最终在江南大学冲过终点线。太湖佳绝处，最美在滨湖，这条浓缩了秋日时光里最美无锡风情的赛道，为所有中外跑者创造了一段最美的回忆！除了优美的赛道和热情的氛围之外，蠡湖半马今年延续了此前的高标准、高品质的服务。滨湖区文体旅游局负责人在接受媒体采访时表示：“今年为了让所有的参赛跑友都能一如既往享受到尊贵客人般的礼遇，各单位、部门全力配合，汇跑赛事专业团队仔细运作之下，最终呈现出了让13000名参赛选手都满意的赛事体验。”今年的赛事共招募和培训志愿者1364名，其中来自江南大学的学生志愿者1074人，社会招募志愿者255人，赛事专员35人。为了确保每一名选手都能得到专业的安全保障，百禾懿启救援中心派出118人参与急救服务，其中36人配备移动AED急救设备；卫健委派出30名医护人员和6辆监护急救车；市红十字会派出30名医疗志愿者参与医疗站服务；组委会配备救援直升机1架，各单位携手同心为选手的安全保驾护航。作为无锡的"优质增长极、活力创新源、美丽湖湾区、服务新高地"，滨湖区拥有无锡90%的太湖湖岸线，汇聚了全市近80%的旅游资源。坐拥如此得天独厚的自然条件，蠡湖半马也成为了无锡市滨湖区深度融合文体旅发展的先锋。今年的蠡湖半马主题为“穿越蠡湖，最美秋日”，在推动全民健身蓬勃发展的同时，更彰显出蠡湖的山水魅力和深厚的人文底蕴，促进体育、文化、旅游三者深度融合，通过多文化形式之间的交流与碰撞，为参赛跑友打造出了一场体验丰富、印象深刻的文体旅嘉年华。滨湖作为体育强区和旅游大区，在打造“高品质美丽湖湾区”的道路上，通过蠡湖半马这项金牌赛事来为这座城市增添新的发展活力，进一步提升滨湖的影响力。与此同时，蠡湖半马这项全民参与的路跑赛事，也进一步带动了滨湖区健康中国建设，让更多市民迈开脚步，加入到健身行列中来。今年的蠡湖半马也得到了滨湖区各大科技产业、新兴产业等企业单位的积极响应，药明康德、雪浪环境、世芯电子科技、十一科技、卓胜微电子等各大企业纷纷组团参赛，充分展现了企业凝心聚力、创新创业的团队精神和滨湖区科技引领、产业强区的发展理念。同时，作为文化、旅游与体育融合发展的代表，市文旅集团也积极组队加入迷你跑团，充分体现了“体育+旅游”以及“诗和远方”的完美交融。滨湖区各镇、街道还自发组织特色团队及拉拉队为选手们加油打气，让来自各地的参赛选手感受滨湖的多彩魅力和滨湖人民的热情！2019比佛利环蠡湖国际半程马拉松已经正式落下帷幕，但走在高质量发展道路上的滨湖区将继续加强文体旅深度融合，全力打造“四季有花、随处见景，规划一流、特色鲜明，文化荡漾、品牌响亮，产业高端、人才向往，社会和谐、人民安康”的高品质美丽湖湾区，为更多来自五湖四海的跑友、游客呈现出太湖佳绝处的每一处动人瞬间。辰辰爱体育
