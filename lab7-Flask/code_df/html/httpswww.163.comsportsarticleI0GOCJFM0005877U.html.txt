勇士险胜独行侠守住前6 库里20+5+13东契奇30+7+17用微信扫码二维码分享至好友和朋友圈网易体育3月23日报道：NBA常规赛继续进行，金州队（38胜36负）拿到连胜。虽然复出得到30分和17次助攻，但得到20分、5个篮板和13次助攻，他率队6人得分上双，勇士队在关键时刻连得4分确立胜局，勇士队在客场以127-125险胜达拉斯队（36胜37负）。勇士队拿到2连胜，独行侠队遭遇2连败。：勇士队的库明加得到22分，库里得到20分、5个篮板和13次助攻，普尔得到16分和6次助攻，追梦得到14分、4个篮板、8次助攻和4次盖帽，迪文琴佐得到14分，卢尼得到12分和12个篮板。独行侠队的东契奇得到30分、7个篮板和17次助攻，得到27分和5个篮板，伍德得到19分和4个篮板，布洛克得到15分，鲍威尔得到13分和4个篮板，格林得到12分和4个篮板。独行侠队的东契奇复出，欧文缺席。开赛后勇士队反客为主，他们以6-2、12-9领先。鲍威尔独取5分，东契奇也有4分，他们率队打出13-2的反击高潮，独行侠队以22-14反超。格林、库明加和库里合取7分帮助勇士队追近，两队之后各打成几次进攻，Ja-格林补篮得手，首节结束时勇士队以30-33落后3分。哈迪跳投命中开启第二节，普尔跑投命中率队连得6分，勇士队以36-35反超。伍德还击5分，东契奇三分命中，独行侠队迅速夺回领先权。之后两队互有攻守，比分交替增加，库明加和库里先后得分，半场前3分55秒时勇士队以51-50反超1分。东契奇罚球追平，汤普森和库明加联手4分帮助勇士队再次拉开，布洛克三分回应，东契奇拿下5分，哈迪突破得手，独行侠队以61-59领先。库里三分命中，鲍威尔空接扣篮，迪文琴佐底角回敬三分，半场战罢勇士队以65-63领先2分。勇士队的库明加上半场得到14分，库里得到9分和7次助攻，卢尼得到8分和10个篮板；独行侠队的东契奇得到18分和9次助攻，伍德得到11分，哈迪得到10分。第三节开始后不久迪文琴佐再中三分，他率队打出7-0的小高潮，勇士队以77-68领先。布洛克手感不错，命中2个三分球，东契奇打3分成功，独行侠队追至79-81。普尔打3分还击，哈迪三分中的，独行侠队紧紧咬住比分。卢尼和普尔拿下4分又扩大优势，格林和哈迪各中三分再次拉近，Ja-格林篮下得分，勇士队以96-93领先3分结束第三节。双方在最后一节开始后各得几分，东契奇和布洛克各中三分，独行侠队追成103平。兰姆成为奇兵，一人独取6分率队打出8-0的小高潮，勇士队再次拉开领跑。独行侠毫不示弱，哈迪连中2个三分球，东契奇突破打进，第四节还有5分20秒时他们追成113平。115平后哈迪跑投帮助球队反超，迪文琴佐三分命中，勇士队以118-117领先。伍德和库明加各罚中两球，哈迪两罚一中追平。追梦打3分成功，勇士队以123-120领先。伍德单打追近，东契奇错失反超机会，库里突破打进，第四节还有8.5秒时勇士队以125-122领先。独行侠队暂停后，东契奇篮下没有打进，卢尼被犯规，两罚全中，勇士队以127-122领先。布洛克投中压哨三分无济于事，独行侠队以125-127落败。勇士队首发阵容：库里、汤普森、迪文琴佐、格林、卢尼独行侠队首发阵容：东契奇、格林、布洛克、鲍威尔、克勒贝尔直播吧
