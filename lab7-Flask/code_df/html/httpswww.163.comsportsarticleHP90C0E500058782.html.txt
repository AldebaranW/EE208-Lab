吕小军回应被禁赛:我无动机也没有理由使用兴奋剂用微信扫码二维码分享至好友和朋友圈网易体育12月23日报道：StartFragment北京时间12月23日，就检测阳性被禁赛一事予以回应。他通过中国举重协会官网发布的声明如下:2022年12月20日我收到了国际检查机构的邮件，通知我的样本检测结果呈非特定物质阳性。我非常震惊，作为一名老运动员，我一直严格遵守国际举重联合会、世界反兴奋剂机构、中国反兴奋剂机构有关反兴奋剂的各项规定。我认为体育的价值就在于公平竞争，我坚决反对使用兴奋剂等各类作弊行为，这也是我一直以来给自己设置的底线，使用兴奋剂完全违背我的价值观，也和我受到的反兴奋剂教育背道而驰。在我24年的举重运动生涯中，我经历了无数次大赛，也接受了几百次兴奋剂检查，没有一例违规。自2022年8月我复出以来，短短4个月时间，我总共接受了兴奋剂检查8次。我非常热爱举重事业，作为一名举重运动员，我已经干干净净的赢得了几乎所有的荣誉，我既无动机也在我挚爱的举重生涯的最后一程使用任何禁用物质。我将配合相关机构展开调查，查出问题的真正原因，还自己清白。12月22日，国际兴奋剂检测机构（ITA）在官方网站发布公告，三届奥运会举重冠军得主吕小军暂时被禁赛，理由是在10月的一次赛外兴奋剂检测结果呈阳性。在ITA最新发布的一份公告中显示，吕小军在今年10月30日的赛外检测提供的样本中，被检出促红细胞生成素（EPO）。EPO被世界反兴奋剂机构（WADA）列入禁用物质清单，规定运动员在赛中和赛外都不得使用该物质。EPO在医学上可用于治疗贫血，可以刺激红细胞生成，并提升体运输氧气的能力，从而增加耐力和表现。在样本检测出违禁物质后，ITA已判定吕小军违反国际举联的反兴奋剂条例，因此暂时将其禁赛，不过吕小军仍有开启B瓶样本复检的权利。ITA是受国际举联委托、独立进行兴奋剂检测的机构，接下来将全权负责此事的调查。今年38岁的吕小军，在伦敦和里约两届奥运会夺得男子77公斤级金牌，其中里约奥运冠军是因为哈萨克斯坦选手尼德札特-拉希莫夫因服用违禁药物而被递补获得荣誉。2021年的东京奥运会，吕小军获得男子81公斤级金牌，成为历史上年龄最大的举重奥运冠军。在巴黎奥运周期，男子81公斤级已经不再是奥运比赛级别，贵为三届奥运金牌得主的吕小军，本计划在明年重返国际赛场，不过现在只能被暂时禁赛。目前，中国举重协会则尚未对此事发表公告。EndFragment梅花说娱乐 2023-10-31 19:42:54沐岭猪猪 2023-11-01 14:14:04阿信点评 2023-11-01 16:51:53钉钉陌上花开 2023-11-01 23:29:31德融视讯 2023-11-01 13:52:10菏泽日报 2023-11-01 09:12:10青岛文艺 2023-11-01 14:55:51历史看点名榜 2023-11-01 10:11:31中国新闻周刊 2023-11-01 13:38:46东球弟 2023-11-01 22:29:30财联社 2023-11-01 09:55:27体坛知识分子 2023-11-01 06:05:03东球弟 2023-11-01 21:43:09讲坛 2023-10-30 16:08:52侃球熊弟 2023-11-01 09:37:23喜欢做饭的苗苗 2023-10-30 00:35:35蓝盛晓梅 2023-10-28 13:40:11西西职场说 2023-11-01 13:22:06东方豪侠 2023-11-01 08:11:10湘楚风云 2023-11-01 23:48:24叮咚体坛 2023-11-01 23:37:56网易体育 2023-11-01 12:36:21体育快播 2023-10-31 19:55:30澎湃新闻 2023-11-01 20:27:13财联社 2023-11-01 15:20:38央视新闻客户端 2023-11-01 15:59:20静海融媒 2023-11-01 14:57:48网易体育 2023-11-01 21:48:58星晨视频 2023-11-01 15:02:49网易体育 2023-11-01 21:31:49澎湃新闻 2023-11-01 16:06:54滑县市场监管 2023-11-01 20:50:41财联社 2023-10-31 22:10:49中国海警 2023-11-01 13:17:44央视新闻客户端 2023-11-01 13:27:42财联社 2023-11-01 15:43:45新京报 2023-11-01 18:14:13不掉线电波 2023-11-01 14:29:21直播吧 2023-11-01 19:20:20不掉线电波 2023-11-01 21:44:48
